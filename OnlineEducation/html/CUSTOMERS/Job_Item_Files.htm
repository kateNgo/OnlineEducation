JOB ITEM FILES

 

This option displays all the details of a particular Job item, Future Service Log and any Warranty details and also enables you to add more details to a particular job item or change existing details about a particular job.

 

Note: A customer can have more than one Job item.

 

TO CREATE OR ADD A JOB ITEM FILE FOR AN EXISTING CUSTOMER:

 

In the AM-WIN window, click the Job Item File icon (5th icon from the left) or from the menu choose Customers, left click and choose, Job Item File. The Job Item Masterfiles window appears.

 

You can locate a job item in a number of ways. You can search by Job item, by Customer, by Group, by Engine, by Chassis, by Make and by Model. Just left click on the relevant tab found across the top of the masterfiles window and simply type on the keyboard the job item code or other relevant information, and the system will automatically scroll and locate the job item file and highlight it (usually in blue).

 

If you are not able to find the job item from the code or by the other tab searches, then you can click on the By Custcode Button or use the F10 key function to choose your customer details. This will allow you to search for a job item by the customer code, name or phone details. Click on the tab required to find the job item and type in the relative information to search by. When the job item is located, click the Select button or press the enter key or double click the mouse to select the highlighted Job item.

 

You can also limit the job item code list by just job items that are owned by a specific customer. To do this you can click on the find by Custcode button located at the top right of the screen or by pressing the F10 key. Select the customer required and then left click using the mouse on the list of job item codes to the left of the screen. This will reduce the list to only job items owned by the selected customer. To get the full list back, blank out the customer code field at the top right of the screen and then once again click on the list of job item codes to refresh the screen.

 

To Create or add a job item file, Click on the Insert button or press the insert key from the Job Masterfiles window.

 

The Job Item code window appears. Enter the Job Item code eg. Registration number or serial number. Click on OK to confirm. The Code will be automatically inserted.

 

Enter the Customer Code of the customer for which a job item is to be created or added, by clicking the Cust button under the Job Item Details tab. A Customer window appears detailing the list of customers.

 

To find the customer code, left click anywhere in the Customers window and type a word or part of a word relating to the code you want, or left click the Customer to be used from the Customers list and this highlights the customer code.

 

Choose Select and the details of the customer will be automatically inserted into the relevant fields in the job item masterfile.

 

Enter the Bill to customer code to whom the bill is to be sent to, by clicking the button on the left of the toÂ field. The Customers list window appears. Left click the customer to whom the bill is to be sent. Choose Select and the customer will be automatically inserted into the Âbill toÂ field. The contact name and phone will also be added automatically when a bill to customer is selected.

 

Update Delivery address as per Customer Â you can tick this box if you wish the delivery address for this job item to be the same as the customer masterfile address.

SMS Button Â Click on this button and the SMS Message Manager screen will appear. Click on the Customer button to select which customer you want to send an SMS to. You will then be able to access these standard messages through the Predefined Message listbox in the SMS Message Manager Screen. Just select your message by double clicking on it, your message will jump to the top message box, activate the reply to option (if needed), choose your phone number and then click on the Send SMS button.

 

To enter in the other details for each Job item, Use TAB or place the cursor in the relevant field to enter the item details for Body, Fuel, Trans, Colour, Radio Code, Eng Cap, Status, Chassis, Engine, Key Numbers, Fleet #, Cyl, A/C and Group.

 

Note : The labels of the fields are user definable and may be changed appropriately from the Tools menu and choose Modify System Field Labels. The Change System Labels window appears. There are 15 user definable labels that can be configured by the user for there specific needs for information storage. This means that the job Item code can be called Rego Number or Serial Number or ID number etc or the chassis number for a car can be named hull number for a boat or landlords name for a plumber. These names are user defined from the Modify System Field Labels file. Enter in the details as required for the user definable fields you have configured. Enter the new labels required in the Job Item Details Screen. Select Save. The labels can only be changed by those users who are authorised to do so.

 

Job Item Status - In the Job item Masterfiles under the Job item Details tab, there is a new field called status. The default status of an item would be Active. If the status is set to suspended then the selected Job item will not appear in the invoice selection window when searching for a job item record unless the status is changed back to active.

 

To view or change customeraddress and contact details click on the Customer Button or use the F9 Button. A Customer details window appears. Change customerdetails where relevant, then select OK to save the details.

 

Spec Code Button Â This provides a link to access a specification database for each job item you have in the Job Item Masterfiles. Inside this specification database are part numbers for such things as Engine Timing, Gas Emissions, Brake Pad F/R, Shocks F/R, Airconditioner belt, Powersteering Belt, Alternator Belt, Spark Plugs, Batteries and other accessories associated with that particular make and model. There are also sub groups for each make and model which includes such things as Fuel filters, oil type, Oil filter, Tyre size, air filter, Coolant type, transmission oil and Air Con Gas.

 

To enter a new spec code into your database, click on the Spec Code button and the Browse the Job item Specs Database window appears. Then click on the Insert Button and name your spec code (eg DawooLanos) and then enter in all the part details for that make and model in the job item specs tab and the Sub Grp and Notes tab, (eg in this case all the specs for a Dawoo Lanos), then click on the Save Button.

 

To select a Spec Code to be used for a job item you have in your system, simply click on the Spec Code button, use your arrow key to scroll down to find your relevant spec code. Then click on the Select button. That particular Spec Code will then be allocated to that job item.

 

URL Link Â This link takes you to a saved folder in your computer where you can save and view documents, images and other files in relation to this particular job item at any time.

 

War. Expiry Button is where you record the date the warranty expires on a job item or part of a job item. Eg, 10/11/09 for a Fan Belt.

 

ENTER SERVICE LOG/OFFSET A/C DETAILS:

 

Service Log - Select the Service Log /Offset A/C tab and left click. To enter Service Log Details, click on the Insert Button. The Vehicle Service Log window appears. Choose the Service Details by clicking on the Service Details button and selecting the relevant description. Click on Select to insert the details in the Vehicle Service log screen. Enter in the meter reading (if applicable) and choose the due date. Click on the Code button to choose which letter you wish to send out for this particular reminder. Click on select to insert those details into the Vehicle Service Log screen. Then click on the OK button. These details will be recorded in the Job item masterfiles screen under the Service log option.

 

Note: The last service details for the job item are Automatically inserted when a service is done to this job item.

 

Service Details Â Enter the service type for the job item such as Service, Overhail etc.

 

Meter Readings Â Enter meter reading for the job item in Kilometres or Hours. Leave this blank if not applicable.

 

Due Date Â Enter date due for service of the job item (automatically inserts Todays date, therefore, change it if needed).

 

Code Â Enter a letter code to send a letter advising a customer that a service is due by browsing the Letter file. To browse the letter file, Left click the Code button on the right of the code field. The Browse Letter file window appears. Click on the list and type the code or part of the code you want. If you can see the code, simply enter or left click on it and this highlights the code which will be automatically inserted in the code field, then choose Select.

 

The Update Button can be used to update the information stored in the Vehicle Service Log Screen. To change the details in the Vehicles Service Log, highlight the relevant entry you wish to change by clicking or moving the up and down arrow keys to the entry you want to change. Choose Update. The Vehicle Service Log window appears. Change the details and to save, select OK.

 

The Print button allows you to print out a service due letter for that job item.

 

Delete button will delete a highlighted job item service reminder. To delete an entry in the Job Item Service Log, highlight the entry you wish to delete by clicking or moving the up and down arrow keys. Choose Delete. A dialog window appears asking if you wish to delete the highlighted record. Select Yes to proceed or No if you wish to abort the deletion.

 

Service Log Notepad - To Enter any other information about the vehicle you can use the notes space provided at the bottom of the screen by clicking anywhere in the Service Log Notepad and entering your details there.

 

Delivery details are also recorded in the Contact, Address, Phone and Fax Fields.

 

THEN:

 

Choose Save in the Job Masterfiles window to save and add the job item for this customer.

 

Choose Clear, to clear the screen (only when creating/inserting a job tem file).

 

Choose Delete in the Job Masterfiles window to delete a job item for a customer. A window appears asking if you wish to delete the highlighted record. Select Yes to proceed with the deletion. Enter any information or notes about the deletion. Select Close. The Job Item is deleted/removed from the Job Item list.

 

Note: If a job item is deleted, all of the itemhistory including invoice and service histories are deleted. A record of all deletions can be accessed by browsing By Code or Entry Date from the main menu, select Others > Logfile for all entries deleted or changed.
