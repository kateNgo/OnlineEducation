Supplier Masterfiles

 

SUPPLIERMASTERFILES

 

This option allows you to add, revise or delete Supplier information. A supplier can also be added at the time of purchasing if a supplier does not already exist on file.

 

Locating a Supplier

 

You can locate a Supplier in a number of ways. If you know the Supplier code or close to it then Left Click on the list of suppliers at the top of the masterfiles window so that one of the supplier codes is highlighted (usually in blue). Then simply type on the keyboard the Supplier Code or part of it and the system will automatically scroll to and locate the supplier.

 

NOTE on colour coding:

Suppliers with a balance owing will be highlighted in the lookup screen in Red

Suppliers with a negative balance will be highlighted in the lookup screen in Blue

Suppliers which are suspended are in Grey.

Normal active suppliers are highlighted in black.

 

This colour coding is very useful to find suppliers that have an outstanding balance, especially when making payments.

 

If you are not able to find the supplier from the code then you can click the Supplier FIND button. This will allow you to search for a supplier by the code, name and phone number as well as displaying the supplieraddress.

 

If you are still not able to find the supplier then you can click on the Binocular button at the top middle of the masterfile screen which will allow you to do an intellisearch for the supplier. Click on the intellisearch link to find out more.
