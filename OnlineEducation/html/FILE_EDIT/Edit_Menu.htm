Edit Menu

EDIT MENU

 

The Edit menu is used to cut, copy and paste information from one area to another, as well as to change the user of AM-WIN.

 

CUTTING, COPYING & PASTING

 

If you highlight information/text within AM-WIN, you have the option to cut or copy it and then paste it into another field. For eg: to save you time entering in a customers address, you may go to Customer Masterfiles, find the address, copy it, and then paste it into the field you want it to be in.

 

Cutting Vs Copying:

 

The difference between cutting and copying is that when you cut you remove the text from its origin. You may then paste it somewhere else.

 

When you copy: the text you copy remains in the original field you are copying it from, but you can still paste it to another field.

 

Highlight the information you want to cut or copy.

 

Go to the Edit menu and choose cut or copy depending on what you are wanting to do.

 

Then, place your mouse cursor in the field or area you want to paste this information in to.

 

Go to the Edit menu and choose paste. The information should now be displayed in the field.

 

Instead of going to the Edit menu to perform these functions, the following keyboard shortcuts perform the same operation:

 

CTRL button + C held down at the same time will COPY

 

CTRL button + X held down at the same time will CUT

 

CTRL button + V held down at the same time will PASTE

 

Note: in order to use these functions, text must be HIGHLIGHTED. If nothing is highlighted, the cut, copy and paste words will be disabled in the Edit menu so that they cannot be used.
