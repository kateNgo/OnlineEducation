LOCATION

 

Under the File Menu from within AM-Win, you can select the location you wish to change to.

If you are licenced for multiple locations, you can select any location that your user profile is allowed to access.

If the location you wish to change to is not available, then you may need to either set up additional locations or alternatively assign your user to the required location.

You can see your current location on the top menu bar.
