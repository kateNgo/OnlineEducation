Printer Setup

 

TO SET PRINTER SETTINGS:

 

Go to the File menu and choose Print Setup. The Print Set up window appears.

 

From this window you can choose whether you to want to print to your Default Printer or to an alternative Specific Printer. By left clicking on the down arrow next to the Name Printer field you can choose from any available printers you may have.

 

Note: you must also set printers up in the Control System. Please see the Control System section in this manual.

 

You may also choose whether you want the printer to print vertically (portrait) or horizontally (landscape). Refer to your own printer manual for explanations of the other options available in this window.
