HIDE / UNHIDE TOOL BAR

 

Many of the users of AM-WIN prefer to use the menu options at the top of the menu screen rather than using the shortcut icons. If you are one of those people then you are able to hide the shortcut tool bar either temporarily or permanently.

 

You can hide it temporarily by selecting the File option from the menu screen and then select the hide/unhide tool bar option. To unhide the tool bar simply select the File > Hide/Unhide Tool Bar option and the tool bar icons will again be displayed.

 

If you wish to permanently hide or unhide the tool bar icons you can do so from the Others > Control System > System & Company Details option. You will need a Password level of 99 to access this option from the control file if you have password security activated. Once in the control file screen you need to click on the Security / Options TAB and tick the Hide the main menu tool bar option on the left hand side of the options screen under Employee & Security heading. You can unhide the tool bar by un-ticking this option.
