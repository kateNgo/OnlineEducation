Text Font Size

 

Screen Font Size

 

Some of our users find that the default size of the text used on an invoice or purchase order is too small, and they manually increase the font size for each block of text on the document or within a kit. There is an option under the ‘File’ menu to customise the default text font. The default font is size 8. You can also customise the font style and make the font bold or underlined if desired. Be aware that if you make the font size too large, the text will overflow beyond the end of the document page and thus be missing off your document.

This feature is available from ‘File’ > ‘Text Font Size, under the ‘Font Control’ tab. To select a font, click on the ‘Font’ button and make your selection, and then click ‘Save’ once you are finished.
