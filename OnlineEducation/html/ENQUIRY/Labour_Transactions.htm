Labour Transactions

 

LABOUR TRANSACTION

 

 

Labour Transaction is used to view labour transactions for a particular customer, within a certain date range, for a particular GL code, or for a particular Invoice number.

 

Go to the Enquiry menu and select Labour Transaction. The Labour Transaction window will appear.

 

To View by Date Range

 

The main window will automatically open at viewing the transactions by date range. This window shows all the labour transactions which have occurred during the Date Range displayed in the From and To fields.

 

To Change the Date Range, left click in the From and To fields and change the dates.

 

Left click the Refresh button. The window will now display Labour Transactions during the new date range.

 

To search by Customer

 

Left click on the down arrow next to the Selection field. A menu should pop down.

 

Select Customer.

 

The Customer list will appear:

 

Select the Customer you want, and left click the Select button.

 

All the Labour Transactions for that customer will now appear in the Labour Transaction window.

 

To View by Invoice Number

 

In the Selection field, choose Inv Number. An extra Inv Num field will then appear. If you press the button next to the Inv Num field the By Invoice or By Code window appears:

 

Highlight the invoice number you want and click the Select button.

 

The Labour Transaction window will now display the labour transactions for that invoice.

 

To View by GL Code

 

In the Selection field, choose code.

 

The By Invoice or Code window appears (as above).

 

Select the Code you want.

 

The Labour Transaction window will now display the labour transactions for that invoice.

 

If you click the Print button, a print preview will appear.

 

Click the Printer Icon to print.

 

The Clear button clears all the labour transactions being displayed.
