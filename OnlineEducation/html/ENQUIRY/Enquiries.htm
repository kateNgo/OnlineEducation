Enquiries

 

ENQUIRIES

 

CUSTOMER, SUPPLIERS, GENERAL LEDGER

 

 

This option enables you to make enquiries on a customer or supplier account as to any money outstanding and for what periods. The option also provides for the status of a particular general ledger.

 

To make an enquiry:

 

In the AM-WIN window, click the Enquiries icon (the 11th icon from the left) or from the menu choose Enquiry, left click and choose Enquiries, left click.

 

The Enquiry on Customers/Suppliers/Ledger window appears.

 

Customer & Supplier

 

To make an enquiry of a customer, select the Customer tab.

To make an enquiry of a supplier, select the Supplier tab.

 

Code - Enter the Customer/Supplier Code by left clicking on the button on the right of the code field. The Customers or Supplier window appears.

 

Enter Customer/Supplier Code or select the customer/supplier by left clicking on the customer/supplier. This highlights the customer/supplier for which an enquiry is being made. Choose Select. The details of the customer/supplier will be automatically inserted.

 

Select The Period

 

Prev Year  Selecting this option will allow you to view the enquiry details for the selected Customer or Suppliers account for a previous year. This option will allows you to view back up to 7 years prior to the current year. A window will be displayed allowing you to select the financial year (only years where data is available will be displayed). You can then select the month of that year to be viewed or select ALL to view the whole financial year.

 

Year  Left click in the circle if you wish to view a list of all transactions for the current financial year for the selected customer / supplier.

 

Month - Left click in the circle if you wish to view a list of all transactions for the current month for the selected customer / supplier.

 

Month Selection  Left click on the menu and select a particular month (July-June). This will display details of a particular month for a customer/supplier. If you have selected Year previously then all transactions from the selected month to the current month will be displayed.

 

General Ledger

 

To make an enquiry of a particular ledger select the G_Ledger tab.

 

Code - Enter the Ledger Code by left clicking on the button on the right of the code field. The General Ledger window appears.

 

Enter Ledger Code or select the ledger by left clicking on the ledger name/code. This highlights the general ledger for which an enquiry is being made. Choose Select. The details of the general ledger will be automatically inserted.

 

Select The Period

 

Prev Year  Selecting this option will allow you to view the enquiry details for the selected Ledger account for a previous year. This option will allows you to view back up to 7 years prior to the current year. A window will be displayed allowing you to select the financial year (only years where data is available will be displayed). You can then select the month of that year to be viewed or select ALL to view the whole financial year.

 

Year  Left click in the circle if you wish to view a list of all transactions for the current financial year for the selected Ledger.

 

Month - Left click in the circle if you wish to view a list of all transactions for the current month for the selected Ledger.

 

Month Selection  Left click on the menu and select a particular month (July-June). This will display details of a particular month for a Ledger. If you have selected Year previously then all transactions from the selected month to the current month will be displayed.

 

Choose Refresh to update the details on the screen when a new enquiry of a customer/supplier/ledger is being made.

 

BUTTONS

 

Print Button  The print button only becomes available when there are records displayed in the enquiry window for the selected Customer, Supplier or Ledger. If records are available, they can be printed by clicking on the Print Button. This report will also show carried forward and accumulated balances for the selected account which are not seen on the screen.

 

Payments Button  This Button directs you to the Open item payments system for the selected customer or supplier. This option is not available in the Ledger Tab and is replaced by the Cheque Button.

 

The Cheque button under the ledger TAB directs you to the cheque entry screen for the selected ledger account.

 

Master Files Button  This button opens the Master file record for the selected account. See the Masterfile Help for this section.

 

Trans F9 Button  The Transaction  F9 button opens a window which contains a full set of transactions that relate to the selected transaction. First select a transaction at the bottom of the enquiries screen and then click the trans button or press the F9 Key.

 

AM-WIN now has the ability to display all transactions which relate to any other transactions for a particular process. Meaning that if your accounts donbalance, or you are looking for the other side of a journal or a cheque the system will be able to tell you the other side of the entry/s and also find the transactions which cause the general ledger to be out of balance!

 

Within Enquiries, find and highlight the transaction you are curious about. For example, you may have done a journal entry where some of the money went into the 1020. You may be wondering where the rest of the journal entry went. By clicking on the Trans button (located on the right hand side of the window) while this journal entry transaction is highlighted a new Transactions window will come up displaying all the other transactions related to that entry (ie: what ledger accounts the rest of the money went to and how much money).

 

Changing details of a Transaction.

 

After selecting the Trans - F9 button you can alter certain details of the selected transaction by clicking on the Change button. For example if you select an invoice you can change the date and description of the selected invoice. You can also add an order number and meter reading from the history module using the F4 key. If you have selected a Cheque or Journal or payment then the date and cheque/ref number can be changed. There is no audit of these changes and due care must be taken.

 

NOTE: Changing the date here does not change the accounting or GST period for this transaction. You should only use this function to correct mistakes.

 

Accessing Copies of Documents from Enquiries Screen

 

Within the enquiries screen you are now able to view/print invoices, credit notes, purchase orders and purchase credits from Customers or Suppliers.

 

Choose either customer or supplier, and select whether you want to view transactions for a specific month, or the whole year. Click on the button next to the Code field and select the specific customer or supplier you want to view transactions for. Once all the transactions are displayed, you simply need to double click on one of the transactions to view them, however it must be an invoice, credit note, purchase order or purchase credit.

 

Once you have double clicked on one, it will open up the standard print options window used in job invoicing etc. Choose your options and press the Go button.

 

Reprinting an Invoice from the Customer Enquiries Screen

 

Within the Customer Enquiries screen and once you have selected a customer within enquiries, you can double click on an invoice and the full invoice will automatically be displayed for viewing or printing if you choose to print it. You can also FAX invoices and other documents directly through your fax modem if you have a fax modem installed and it is correctly configured.

 

Accessing Purchase History from Supplier Enquiries Screen

 

Within the Supplier Enquiries screen, there is a History button on the right hand side.Once you have selected a supplier within enquiries, pressing this History button will open up the Purchase History for the selected supplier automatically- showing all purchases from this supplier and details of the purchases within a date range.

 

Reprinting a remittance advice from the Customer or Supplier Enquiries Screen

 

Within the Customer or Supplier Enquiries screen and once you have selected an account within enquiries, you can double click on a supplier or customer payment and the full remittance will automatically be displayed for viewing or printing if you choose to print it. You can also FAX remittances and other documents directly through your fax modem if you have a fax modem installed and it is correctly configured. This is great for seeing what invoices were paid on the selected payment.

 

Reverse Button  This button reverses the date order in which the transactions are displayed. Usually the most recent transaction is at the top. By clicking on the reverse button the transactions will be displayed in chronological order from oldest to newest. This order is also the way the transactions are printing. It is usually a good idea to reverse the order of transactions prior to printing them.

 

Job Invoice Button  This button from the Customer TAB allows you to open a new invoice for the selected customer directly from the enquiries screen.

 

LIST BOX SORTING.

 

This is a handy feature that allows you to sort the enquiry transactions into either Source, Date, operator, description, reference or gross value order. From the enquiries screen for customer, supplier and ledger you can now click on the gross column and the list of records will be sorted in order by the dollar amount. Clicking this menu button again will sort the list in reverse dollar amount. This is great when looking through a large file such as all entries in the bank account for a 12 month period and searching for a dollar amount of a cheque or other entry. Simply click on the heading from within the enquiries window. Refer to the Sorting information section for more details.
