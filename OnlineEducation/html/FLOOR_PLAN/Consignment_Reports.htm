Consignment Reports

CONSIGNMENT REPORTS

 

Go to the Consignment menu and go down to Consignment Reports, and select the desired report.

 

The Sold Unpaid Report displays a list of consignment stock which have been sold but have not been paid for as yet.

 

When this report is chosen from the menu, the Consignment Report Options window will be displayed.

 

You may change the date by using the up and down arrows

 

Click on the Go button to see a preview of the report.

 

The On Hand Report displays a list of what stock is ready to be sold and details about the item.

 

The Consignment Report Options window will be displayed.

 

Use the up and down arrows to choose the date you want the report to be up to and then click on the Go button.

 

This report displays the sales history of all Consignment Stock within a range of dates.

 

When this report is chosen from the menu, the Consignment Report Options window will be displayed.

 

Use the up and down arrows next to each date field to select the date range.

 

Click on the box next to the Show Details on the Report

 

Click on the Go button to see a preview of the report.

Returned Goods

 

This report displays all items which have been returned.

 

When this report is chosen the Consignment Report Options window will be displayed.

 

Use the up and down arrows next to each date field to select the date range.

 

Click on the box next to the Show Details on the Report

 

Click on the Go button to see a preview of the report.
