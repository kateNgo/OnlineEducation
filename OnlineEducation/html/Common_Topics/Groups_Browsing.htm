Groups Browsing

Browsing the Groups file

 

Left click the button on the right of the group field. The Browse the Group file window appears.

Note: Groups can be added/changed/deleted from Groups Maintenance in the Others menu.

Left click anywhere on the list and type a word or part of a word which refers to the group code you are searching for. Or simply enter the group code if you know it or left click the applicable Group and this highlights the group code to be automatically inserted in the group field. Then choose Select.

 

Note  Use the up and down arrow keys or scroll down for a complete list of the group.
