CREATING A HIRE

 

A hire can be Permanent or Casual. A Permanent Hire might be for a 2 year period and can only be based on weekly or monthly price rates.

 

Casual booking would be based on an hourly, daily or weekly rate.

 

To create a Hire, click on the Booking/Invoice icon, which will open up the Hire Invoicing screen.

 

Firstly, click on the down arrow next to the Type field, and select either Casual or Permanent. Depending on your choice, the relevant current bookings will be shown on the screen.

 

Then choose whether the Hire will be monthly, weekly, daily or hourly.

Select the Customer by clicking on the Customer button.

 

You will now have access to the Hire button. Select this button.

 

After selecting the Dates required for the Hire, you will then be shown the Hire items which are available. By clicking on the Availability Search button, you can check if the highlighted Item is available for the required period of time. Select the required Item.

 

The Availability screen will then appear to show available time/dates. If the availability suits the hire, click on the select button. If not, Exit the screen and choose another Item.

 

The Insert Hire Record screen will then appear with the relevant rates etc detailed. The Contractor name can be added if required by clicking on the Contractor button.

 

Adjust the Insurance value if necessary and this will be auto calculated. Click on the OK button.

 

The details will then be entered on to the New Hire screen.

 

From here any changes or additional details can be added to the invoice eg., additional parts which may have been used or additional misc charges. These charges are only relevant to casual hire. The Levy button will enable you to charge a Levy which would be appropriate for your business. You will need to select a ledger and nominate the required percentage for the levy and the system will calculate the value required on the invoice.

 

Click on the Save button.

 

From here you can then print an Estimate or print a checklist or agreement as required.

 

NOTE: When creating a Permanent hire, you will be required to setup the next invoice date. This would normally be the 1st of the following month, in that the initial invoice would generally contain additional charges which are one off costs, which would not be included in the normal monthly rental invoices. You would then do the first invoice as a casual rental, then, when complete, change the hire to Permanent for the monthly rental only.

 

If you then go to Programs>Hire Events you will note that this Permanent Hire has been added to the list.

 

If you then go to Programs>Autobill, you can view the invoices which need to be processed for the period.

 

The Due Date is always the last day of the following month. The Process date will be the date of the invoice.

 

By clicking on the Process button, the invoices will be processed and printed

 

Note: A Permanent Hire can be changed to a Casual Hire if arrangements have been made with the client to retrieve the Item for a short period of time. A credit note can then be raised for the period of time the Item is charged out on a Casual basis.
