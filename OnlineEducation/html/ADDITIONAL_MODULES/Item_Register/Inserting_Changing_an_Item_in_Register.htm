Inserting/Changing An Item in the Register

 

The Main Item window allows you to add 3 types of Items. These are Internal, For Hire, and For Sale. These are all indicated by buttons at the top left of the screen including another button labelled All. When an Item has been created it will appear in the Item list below these buttons. The Items being displayed in this window will depend on the Item Category button selected unless a Filter is used to refine the Item list to just those Items which match the Filter criteria. If you press the Internal Button then only the Internal Items will be listed. If you press the For Sale button then only the For Sale Items will be listed and so on.

 

You can access the Item register screen in a number of ways. You can either click on the blue filing cabinet icon at the top left of the screen or select the Item Register option under the Programs menu.
