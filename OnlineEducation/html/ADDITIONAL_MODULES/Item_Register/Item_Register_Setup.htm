Item Register Setup

 

When the Sales Module and Item Register are installed, there are certain settings which need to be configured.

 

The Item Register automatically has ledger numbers inserted into the program to allow the figures relating to sales and expenses to go to the correct ledgers. These ledger numbers will need to be checked in your General Ledger Masterfile. If any of these numbers are currently being used, you will need to create a new ledger for the pre existing code and journal the balance over to the new ledger code, then rename the ledger with the appropriate GST Type as listed below.

 

Alternatively, you could change the ledger code in the Item Register in the Ledger/Expenses tab, as long the ledger is still in the nominated ledger code series as shown below. To change the Plant and Equipment ledger from 1100 to another number will need to be done through the Supervisor button in the Item Register. This is not advisable as the system will still continue to use the default numbers and as such generate a lot of extra work and also increase the chances of errors if you forget to change the codes. If the default codes do not currently exist then they will be automatically created when you open the AM-Win Version 7.6 or later program for the first time.

These ledgers are:

 

1100  Plant and Equipment  - Acquisition

1101  Provision for Depreciation  - N/A

5900  Item Labour Offset  - Gst

6100  Item Income   - Gst

7100  Item Sale Cost   - Gst

7900 - Item Maintenance Offset - Gst

8100  Insurance Recovery  - Gst

9100  Depreciation Expenses - N/A

 

In the Control System you need to nominate a few options in the Security Tab and then Field Controls.

 

Management Control  If you have Password Security turned on in Am⌀win, depending on the Password Level entered into this field, certain areas will be inaccessible to anyone with a Password Level lower than what is set, relating to invoicing and the Item Register, such as recosting jobs (total button), accessing cost and sell prices in customer invoice.

 

You can also assign the next Item number or automatically assign item or numbers to your items. We recommend you start with an item number of 10,000. This is done under the Control System in the Invoice tab.

 

Item Register - Enter the Password Level required to gain access to the Item Register.

 

Disable the Updates in the Item Register  If you have Password Security turned on in Am-win, enter the password level into this field to disable changes to be made in the Job Item masterfile for any items relating to the Item Register.

 

Item Category  In the Item Register, choose the option which relates to how you are using the Item Register, whether you are using the Item Register or hiring, sales or for internal Item control only.

 

Sales Order Module  Choose which industry your business relates to, so that the labels showing in the Sales Module are showing the appropriate information.

 

Insert Parts to Jobs from Sales Module  If this option is ticked and a jobcard is printed from the Sales Module the parts will automatically be inserted onto the jobcard when the jobcard is created. This would only be relevant in the case of the sale being done between internal departments within your own business, such as between the sales dept and workshop.

 

In the Sales Module, under the heading of Programs>Sales Order Options, you are required to enter the State you are in and your Dealer licence number. Fill in any other details on the screen, which would be applicable to your business.

 

You must also enter the internal customer that you wish the owner to be for all of your items. This default owner must be internal and is very important to ensure the item sales module works correctly.

 

Details such as stamp duty, luxury tax etc must also be entered if they are applicable to your state. These settings vary from state to state. An asset can also be marked as being exempt from these taxes depending on the industry and state you are in.

 

Also under the heading of Programs>Industry, if the required headings are not already set for your industry or if you wish to change any of the headings, it can be done from this screen. These headings will relate to the information on the Item, which will need to be maintained.

 

The marine industry settings are provided with the software and must be used in this format to allow easy and trouble free exporting of the items to our sales module website or to the trading post or other advertising option.
