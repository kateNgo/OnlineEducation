Print Invoice Completed Buttons

 

Print Button

 

The Print button when clicked brings up a list of print options for you to choose from. At this point you can elect to print a Sales Contract, Invoice, Quotation or Proforma Invoice in relation to a sale. To select which report you would like to print just click in one or more of the small boxes which should insert a small tick next to the item you wish to print. Then click on the OK button. If you wish to print a proforma invoice or call the document by another name, you can manually type in the field below the print proforma invoice option what you would like to name that particular document, ie Estimate, Tax Invoice, Proforma Invoice, Cash Sale etc. A Tax Invoice can only be printed once the invoice has been finalised

 

Invoice Button

 

The Invoice button when pressed will automatically categorise the current order from a sales order to finalised. This means that the current sales order will now be processed into the financial figures and all associated accounting and GST data is processed. You must also process the Trade-in details separately which can be done either before or after finalising the sale.

 

When you process the Trade In a screen will appear showing the relevant details and the Item will be added to the Item Register.

 

At this stage the system allows you to automatically book the trade-in into the workshop to be checked for resale unless you untick the workshop job option.

 

Completed Button

 

The Completed button when pressed will automatically categorise the current order from a finalized or delivered order to Completed. You can only complete an order if the sale is finalised and all payments are processed and the sales order has a balance of Zero on the bottom right hand side. This means that the current sales order has now be processed into the current financial figures and all payments owing on the sales order have been made and nothing can be changed on the order.
