Item Accessories

 

When you select a package, the system automatically updates the quotation/sales order with any accessories tagged against the selected Item. You can then add any additional accessories to the quotation through the Accessories button under the Package button on the left of the screen if you are in the summary tab or on the right of the screen in any other tab area. Once the accessories button has been clicked you are then able to add detailed text, spare parts / accessories, revenue items or alternatively either change or delete existing items.

 

The Add Text  F2 button allows you to add text or comments manually into the Quotation. Click on the Ad text button and then type into the text area at the top of the screen. You can then do a spell check using the spell check Quill Button. The Ok  F12 button will then save the entered text onto the Quotation. The Cancel Button will abort the text entry. The Font Button allows you to change the text font to a different colour or to a bold, underlined or other font. You can also change the font name for a different or unusual font to be created. Font changes apply to the entire text currently in the scratch pad area. The J/Notes  F5 button allows you to bring any notes attached to the job item or Item into the scratch pad area. You can then modify this text and place the modified text onto the Quotation. The Preview  F6 Button displays the entered text at the bottom of the scratch pad area as it would appear on the Quotation screen prior to actually adding the text.

 

You can then use the change text button on the bottom right of the tab to alter any text details you have previously entered onto the quotation.

 

The Parts  F3 button allows you to add accessory parts from your spare parts file onto the Quotation. After clicking on the Parts Button the cursor is placed on the part number field. You can then either barcode scan the part number or manually enter either all or part of the stock number and press enter to bring up the list box. You can then scroll down the listbox using the arrow keys and when the blue highlighting bar is over the part you need, press the select button to select your part and it will be dropped into the quotation. Once you are back in the quotation screen, you can then adjust your quantities and press enter to insert them into the quotation.

 

You can also enter Labour using the Labour  F4 button and Revenue using the Revenue  F5 button. The Revenue button allows you to add items other than the parts or labour, which may include sundry charges, freight charges or any other miscellaneous accessories or charges.

 

You can also enter comments into the accessories tab using the Comments button. The comments can consist of up to 10 lines of detail under each comment and may be things such as warranty messages, accessory descriptions or any other messages. You can add the comments through the normal AMWIN program under the Others > Comment Library menu.

 

Kits may also be added using the Kits  Alt K button. A kit contains a full list of text, parts and revenue items and can be used to insert an accessory package such as a safety kit which may consist of a number of different items. Kits can be added under the Others > Kit Library option of the main AMWIN program or under the Kits option under programs in this module.

 

You can then use the change of delete keys to either modify or delete existing information. If you click on a item in the accessories list you can then modify the item and click on the change button to save the modifications. Alternatively you can highlight an item in the list and click the delete button to delete the item. You can also delete an item by using the Ctrl & DEL keys together which will delete the selected item in the list.

 

When all accessories have been added, click on the Save button, and you will be returned to the Order screen.
