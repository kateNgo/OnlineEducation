On Costs

 

The On Cost tab allows you to add any on costs associated with the Item. When you select a package, the system automatically updates the quotation/sales order with any on costs tagged against the selected Item. You can then add any additional on costs to the quotation through the On Costs button under the accessories button on the left of the screen if you are in the summary tab or on the right of the screen in any other tab area. Once the on costs button has been clicked you are then able to add these costs or alternatively either change or delete existing items.

 

The Insert button allows you to insert additional on costs. After pressing the insert key the cursor will be highlighted on the Code column where you can then enter or search for the selected revenue code and then alter the description, quantity or price of the item.

 

The save and delete buttons are then available to save any changes to an existing item or to delete an existing item. You can then use the Ctrl & DEL keys together, which will delete a selected item from the On costs screen.

 

The clear button clears the screen.
