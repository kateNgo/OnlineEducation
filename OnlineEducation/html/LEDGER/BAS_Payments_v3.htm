BAS PAYMENTS

 

There is now an option to pay BAS payments. There is no need for a manual journal entry to be done. Simply enter the cheque, GST amount, installment income if any, PAYG amount, and any fringe benefits tax, and the system will ensure your payment balances and then automatically do the journals required to update the ledger and Bank Rec etc.

Go to the Ledger menu and choose BAS Payment. The BAS Payment window will open.

 

Date: click on the button next to this field and select the date by clicking onto the calender which will appear

Refer/Cheque: type in the reference or cheque number

Method: click the arrow next to this method field and select the method

Description: type in the appropriate description

Amount: type in the total amount which is to be paid. Please note: if the amount is payable this must be a Positive Amount.

It the amount is a refund, it must be a Negative amount.

 

Within the GST, PAYG, Instalment fields, simply type in the amount allocated to these accounts. Please make sure you have set up accounts for GST, PAYG and Instalment in your general ledger masterfiles.

For the Fringe Benefits field, click on the button next to the blank field next to it, and select the ledger account you want the fringe benefits tax to go into.

 

As long as the Left to Post = 0 you may click the Process button to process this BAS payment. AM-WIN will automatically do all necessary journal entries and updates.
