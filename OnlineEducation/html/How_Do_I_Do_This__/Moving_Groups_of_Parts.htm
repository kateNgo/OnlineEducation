Moving Groups of Parts

 

Moving a Group of parts to a Different Group

 

This option is used when you want to move all the parts in one group over to another group.

 

Go to the Tools menu and choose Merge groups in parts file

 

The Merge Group window appears.

 

Click on the button next to the From Group field. A Browse the GROUP File window appears. Highlight the group you want to move all the parts out of, and click the select button:

 

This will return you to the Merge groups window where the From Group field should be filled in.

 

Click the button next to the To Group field to choose what group you want to insert all the parts into.

 

The Browse the GROUP file window will appear again. Choose the group you want to move the parts to, and click the select button.

 

This will return you to the Merge groups screen where both fields should be filled in.

 

To change either of these fields, simply click on the button to the side of the field and select a different group.

 

Click the OK button when you have completed your selections. The merge groups will process and the window will close.

 

Please note that the group you move all the parts out of will still exist, however it will be empty. You can delete this group in Groups Maintenance which is under the Others menu in AM-WIN.
