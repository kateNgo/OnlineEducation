Find Button

 

The find button is located in the masterfiles screens.

 

Customer  You can find a customer by either the customer code, name or phone number by left clicking on the appropriate tab (by Code / by Name / by Phone) and then entering in the required Code, Name or phone number to locate the customer.

 

Supplier-  You can find a supplier by either the supplier code, name or phone number by left clicking on the appropriate tab (by Code / by Name / by Phone) and then entering in the required Code, Name or phone number to locate the supplier.
