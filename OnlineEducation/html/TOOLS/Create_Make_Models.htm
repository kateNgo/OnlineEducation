Create Make & Models

 

Make and Model File

Under the TOOLS menu from within AM-Win there is an option to create Make and Models. Select this option which will open the make and model window. You can type in either the make or model which will allow you to search for the required record.

 

To insert a new make and model file you can click on the insert button or press the insert key. This will open up a new screen allowing you to add the new make, model and year. Once this is done the system will automatically create a unique code for this record that links to a powerful feature linking parts to this make and model record.

 

You can also change or delete a record by clicking on the change or delete buttons. NOTE: that if you delete a vehicle then the parts link to that vehicle will also be lost.
