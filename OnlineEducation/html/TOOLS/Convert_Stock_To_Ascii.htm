Convert Stock To Ascii

 

CONVERT STOCK TO ASCII

 

This option allows you to export your entire stock file to an ascii comma delimited file which can then be imported into other programs such as MS Excel, MS Word etc. It can then be used to distribute your price list to your customers in a format that their computers may be able to import from. The system creates this file in your DATA directory and calls the file stk_ascii.dat.

 

Simply click on the process button to create the file.
