MODIFY SYSTEM FIELD LABELS

 

This option can be used if the field labels in AM-WIN are not appropriate or are not the preferable terms which you use for your business. For example, you may wish to change the Field Label REGO number to Boat number if you are in the Marine industry.

 

In the Tools menu, choose Modify System Field Labels. The following Change Systems Labels window will appear.

 

To the right of each field are suggestions of other field names which you can change the original field name too, however you are free to enter any label you feel is appropriate.

 

Left click into the field you wish to change, and type in the new field label.

 

Left click the Save button once you have completed all the changes. This will immediately update those field label changes in all areas of AM-WIN.

 

Click the Close button to close the window.
