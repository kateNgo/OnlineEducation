Changing Preferred Supplier for a parts Group

 

This option is used to change the preferred supplier for a whole group of parts, rather than only some parts from that group.

 

Go to the Tools menu and choose Chg Pref Supp for Parts Grp.

 

Click on the button next to the Select Group field. The Select stock GROUP Window will appear.

 

Select the GROUP you want by highlighting it, and then pressing the Select button. You will now return to the Changing Pref. Supplier for Parts Group window, where that group code will be displayed in the Select Group field.

 

Click on the button next to the Supplier field. This will bring up a list of suppliers.

 

Select the supplier you wish to be the preferred supplier by highlighting it in this list, and clicking the select button. This will return you to the original window where the Supplier field and Name field will be filled in.

 

Click on the Ok button.

 

The records will be processed, and the window will close.
