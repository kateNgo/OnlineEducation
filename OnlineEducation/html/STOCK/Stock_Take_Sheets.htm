Stock Take Sheets

 

STOCKTAKE SHEET

 

This option prints out all of your current stock levels and allows you to use the sheet to do a manual stocktake and record your actual stock figures. You should use the stocktake section to adjust values in the computer.

 

Go to the Stock menu and choose Stocktake Sheet. The progress window appears.

 

All -  This will print a stock take report for all stock items.

 

Supp  This will print a stock take report for parts purchased from a selected supplier.

 

Jobs  This will print a list of all parts currently on unfinalised jobs in progressed that need to be considered during the stock take.

 

Group  This will print a stock take report of all stock within a specified stock group.

 

Bin  This will print a stock take report for all parts between a stock take range. The system will take a while to access the list of bin locations if you have a large stock file.

 

Alternate  This will print a list of alternate stock items for stock taking.

 

To do a stocktake sheet for a certain group, left click on the button next to the blank field. The Select Stock GROUP window will appear.

 

Select the group you want by highlighting it and clicking on the Select button. The GROUP will then appear in the stocktake sheet window.

 

Suppress stock on hand field  This option if ticked will suppress the printing of stock on hand figures so that the counter will not estimate that the computer is correct and they will have to actually count each item.

 

Ignore if stock qty is zero  This option if ticked will only display items where there are stock on hand for those items.

 

Include Alternate Stock. - This will print a list of alternate stock items with the main stock item.

 

Left click the Go button to see a stocktake sheet print preview.
