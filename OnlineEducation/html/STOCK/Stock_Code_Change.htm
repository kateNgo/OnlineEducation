Stock Code Change

 

CHANGE A STOCK CODE

 

This option is used to change a stock code or part number.

 

To change a stock code:

 

In the AM-WIN window, choose Stock from the menu, left click and choose Code change, left click.

 

The Stock Code change window appears.

 

Enter the stock code you wish to change into the Old Stock Code field.

 

If you are unsure of what the code is, left click on the button to the right of the Old Stock Code field. If you know the old stock code, then proceed to Step 16.

 

The Select Stock or Alternate Stock window will appear.

 

Enter a word or part of a word that is related to the Stock Item Code in the Locator field. This will highlight the stock item you searched for. Choose Select and the Stock Item Code will be automatically inserted in the Old Stock Code field.

 

Click the Description tab. A search can be similarly conducted as in Step 7 but this time using a word or part of a word that is related to the description of the Stock Item.

 

A similar search can also be conducted using X-ref of a Stock Item by clicking on the X-Ref tab.

 

Alternatively, choose Search. The Intelli-Search on Stock window appears.

 

Enter a word or part of a word that is related to the Stock Item or Alternative Stock item in the Containing field. Tick the boxes by clicking the matches to be found in the Stock File or Alternative Stock File. Choose Find Now.

 

Choose Stop, to abort a search.

 

Note: The search may need to be aborted due to the word or part of the word being too insufficient and needs to be further defined for the search to be conducted.

 

Choose New Search, to conduct a new search.

 

If the search is successful, the Stock item/s will appear. Left click on the stock item to be viewed in the Old Stock Code field. This highlights the stock item code, which is to be automatically inserted. Choose Select.

 

This should then return you to the Select Stock or Alternate Stock window.

 

Left click on the Select button. This should return you to the Stock Code Change window where the stock code you selected will be displayed.

 

Once the stock code you wish to change appears in the Old Stock Code field, press TAB on the keyboard or place the mouse cursor in the New Stock Code field and left click.

 

You should now be able to type in the New stock code you wish to have for the stock you selected, into the New Stock Code field.

 

Once you have typed in the new code, left click on the Update button.

 

This will update the Stock file as well as the Sales & Purchase History of the Stock.

 

Once the update button has been pressed, the Stock Code change window will be clear, and you may enter another stock code that you wish to change.

 

If there is no more stock codes that you wish to update, left click on the X which appears at the top right hand corner of the Stock Code Change window.
