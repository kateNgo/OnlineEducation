Alternate Stock Enquiry

 

Under the Stock menu there is an option called Alternate Stk Enquiry. When you click on this option the Alternate Stock window appears

 

This window allows you to do a sales enquiry for alternate stock items.

There are 2 ways in which you can search for the sales information. You can either search by looking up the Alternate Stock code by clicking on the button next to the field Alternate Stk Code. The Browse the Alternate Stock window will appear for you to choose the stock code.

 

Alternatively, you can search for the Alternate stock by looking up a Stock Code by clicking on the button next to the Stock code field. The Select Stock or Alternate Stock window will pop up. Find the stock item whose Alternate Stock you want to enquire about. Once you choose that stock item the sales information for its alternate stock will appear on your screen.

Accessing Alternate Stock Enquiry from Stock Masterfiles

 

On the top part of the screen there is an Alt Stk Enquiry button which when pressed will open up the Alternate Stock enquiry window for the particular stock item which you have selected in Stock Masterfiles
