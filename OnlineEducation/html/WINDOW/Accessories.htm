Accessories

 

Calculator

 

This option will access the standard windows calculator function and is also available at any time by pressing the ALT-F7 keys together. You can cut and paste the calculator value from the calculator into a numeric field in AM-Win by using the CTRL-C copy and CTRL-V paste functions available in most windows applications.

 

Calendar

 

This option will access the standard AM-Win calendar function and is also available at any time by pressing the ALT-F8 keys together.

 

Notepad

 

This option will access the standard windows notepad function and is also available at any time by pressing the ALT-F9 keys together. This will work as a notepad / memo pad area where you can type and save notes independent of AM-Win.
