Invoice Register

 

This report gives you the ability to print out a detail or summary report for all invoices written to all the customers or a particular customer within a specified date range. It also allows you to print a report for a particular job or invoice.

 

The Invoice Register Report Option window will open with a Report Type field and a Print Synopsis of each Invoice option.

 

Left click the down arrow in the Report Type field and choose from: All (all invoices), Customer (invoices of a particular customer), Job (invoice for a particular job),Invoice (a particular invoice), Invoice Range (all the invoices between two particular invoice numbers), Date Range (all the invoices between two particular dates).

Depending on the report type you choose, certain extra fields will be added to the above Invoice Register window, for you to fill in with the following information:

Customer:   type in the customer code

Job:    type in the job number

Invoice:  type in the invoice number

Invoice Range: the two invoice numbers which set the invoice range

Date Range: the two dates which set the date range

 

Tick the box next to Print Synopsis of each invoice for more detail.

Left click the Go button.
