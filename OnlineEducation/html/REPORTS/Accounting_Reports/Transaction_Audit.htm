Transaction Audit

 

This report allows you to print a report showing any out of balance transactions where the transactions do not balance back to zero. This usually occurs when there is a computer lockup or crash or power failure and not all ledgers are posted to. The system will report the out of balance entries so that you can fix them using the Check Tran Masterfiles option. This will also usually mean that the trial balance will not balance.

 

You can select to only show not balanced accounts or to show all accounts including balanced entries between a selected date range.
