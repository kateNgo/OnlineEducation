Jobs in Progress

 

This option is used to report all the jobs in progress

 

In the Work in Progress Report Options window, the date range will default to one month ago from todaydate. To change this, left click in the From and Until fields and type in a different date. Or use the button next to these fields to use the calender to select the dates.

 

If you want the work in progress only for a particular salesperson, type in the salesperson initials in the Salesman field.

 

Left click the Go button.
