Customer Overdues

 

This option allows you to select a particular overdue letter from your letters file, and send this letter to all customers or an individual customer who may be a certain number of days overdue. Leave this field blank if you want to report on all customers.

 

 

When the Overdue Letters Print Option window appears, left click on the button next to the Letter field.

 

A Select Letter window appears.

 

Highlight the letter you want, and either left click the Select button, or double click on the letter.

 

This will return you to the Letters overdue print options window where the letter code and description you have selected will be displayed.

 

Type in the number of days overdue the customer/customers are in the Overdue By field.

 

Left click the Go button if you want to print the letter for ALL customers.

To print the letter for particular customers, click on the button next to the customer field and select a customer from the Select Customer window which will appear.
