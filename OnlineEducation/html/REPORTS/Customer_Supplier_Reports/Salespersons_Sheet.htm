Salespersons Sheet

 

Salespersons Sheet

 

This option is used to print a list of all the salespeople (as entered in the Salesman File in the Others menu) and their territory.

 

This list will automatically print out once Salesperson Sheet is selected from the Customer/Supplier reports menu.

 

Salespersons Sales

 

This report is used to view a sales report for ALL or one salesperson within a specified date range.

 

In the Salespeople Sales Report Option window, use the up and down arrows next to the From and Until fields to select a date range. Alternatively, click on the button next to these fields to use the calender to select dates.

 

You can also enter a sales commission percentage and the system will calculate the commission on your gross profit for each sale.

 

To view the report for particular salespeople, click on the button next to the Salespersons Intials field and select the salesperson from the Select Salesman window.

 

Left click the Go button if you want to view the sales report for ALL salespeople.
