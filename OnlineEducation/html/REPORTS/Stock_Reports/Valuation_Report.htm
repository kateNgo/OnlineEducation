Valuation Report

 

This report gives you the amount of items on hand, the average cost of the item, the last cost of the item, and the value of the item (the average cost multiplied by the number on hand).

 

When the Report Options window appears left click on the arrow next to the Include field and choose from All (a report for all items), Stock (a report for only stock items), Non Stock (a report for only non-stock items), consignment stock, revenue or trade-in. Refer to the Stock Type Topic for a full description of these items.

 

Select the ledger asset code between 1040  1049 to be included in the valuation. For all stock regardless of the asset code select the ALL option.

To insert the GROUP you want included on the report, click on the button next to the Group field and select the group you want from the Select Stock Group window which will come up. The group will then be inserted into the Group field.

 

To insert the Sub GROUP you want included on the report, click on the button next to the Sub Group field and select the group you want from the Select Stock Sub Group window which will come up. The group will then be inserted into the Group field.

 

Left click in the blank boxes for Ignore negative stock and Ignore zero stock if you donwant this stock to appear on the report.

 

There is also an option to print group totals only which will only print the total value of each stock group and not the individual items within that group.

 

Left click the Go button for a report print preview
