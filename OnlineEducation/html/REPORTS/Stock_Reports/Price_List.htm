Price List

 

This report prints out a list of all prices on file. The price list can be printed out for particular groups, or for all items, either stock, non-stock- or both.

 

In the Stock Price List Report Options window, left click on the arrows next to the Include and Sort Order fields and make a selection from the pop down menus. Choose from All (a report for all items), Stock (a report for only stock items), Non Stock (a report for only non-stock items), consignment stock, revenue or trade-in. Refer to the Stock Type Topic for a full description of these items.

 

Select the prices you want included in the price list report by left clicking in the small boxes next to the different price options, so that a tick appears in the box.

 

For a price list for only particular groups, click on the button next to the Group field and select the group you want from the Select Stock Group window which will come up. The group will then be inserted into the Group field.

 

To insert the Sub GROUP you want included on the report, click on the button next to the Sub Group field and select the group you want from the Select Stock Sub Group window which will come up. The group will then be inserted into the Group field.

 

Left click the Go button to see the report which will display the stock codes, descriptions, units, and the prices you have selected to be displayed.
