Stock Sales By Group

 

This report displays the sales of all your items, stock, non-stock or both, or for particular groups.

 

In the Stock Sales Report Options window, click on the arrows next to the Include field and the Sort Order field and make a selection from the pop down menu.

Choose from All (a report for all items), Stock (a report for only stock items), Non Stock (a report for only non-stock items), consignment stock, revenue or trade-in. Refer to the Stock Type Topic for a full description of these items.

 

To select a particular GROUP left click on the button next to the Group field.

 

The Select Stock GROUP window will open where you can select the group you want by highlighting it and then clicking the Select button.

 

To insert the Sub GROUP you want included on the report, click on the button next to the Sub Group field and select the group you want from the Select Stock Sub Group window which will come up. The group will then be inserted into the Group field.

 

By State Only  If this box is ticked then you can select the state (eg: NSW) and only sales for that state are printed.

 

Choose the Report detail level - Synopsis shows a total of each group

     Details shows all of the details of each sale for each

     Group

     Group totals only shows just the total of each group.

 

Click on the up and down buttons next to the From Date and Until Date fields to select a date range for the report. Alternatively, click on the buttons next to each of these fields to use the Calendar to select dates.

 

Left click the Go button to see the report.
