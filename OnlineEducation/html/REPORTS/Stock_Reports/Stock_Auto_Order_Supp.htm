Stock Auto Order - Supp

 

In the Auto-Orders for Supplier Report Options left click on the Include field and Choose from All (a report for all items), Stock (a report for only stock items), Non Stock (a report for only non-stock items), consignment stock, revenue or trade-in. Refer to the Stock Type Topic for a full description of these items.

 

Left click on the button next to the Supplier field. This will bring up a Select Supplier window from which you select the supplier.

 

Click on the Go button to preview the report.

 

The report will display the items for the supplier chosen. For each item it will display the Order amount (the amount to be ordered), the Allocated amount (quantity of that part required for any jobs in progress) and the Suggested quantity which should be ordered:
