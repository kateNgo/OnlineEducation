Redundant Stock

 

In the Redundant Stock Report Options window, click on the arrows next to the Include and Sort Order fields, and make a selection from the pop down menus.

 

Choose from All (a report for all items), Stock (a report for only stock items), Non Stock (a report for only non-stock items), consignment stock, revenue or trade-in. Refer to the Stock Type Topic for a full description of these items.

 

In the Before Date field, use the arrows to select a date so that all sales before that date will be included in report. Alternatively, click on the button next to this field to use the calender to select a date.

Left click in the box next to Exclude Zero Stock to indicate that the report should not include items with a zero quantity.

 

To insert the GROUP you want included on the report, click on the button next to the Group field and select the group you want from the Select Stock Group window which will come up. The group will then be inserted into the Group field.

 

To insert the Sub GROUP you want included on the report, click on the button next to the Sub Group field and select the group you want from the Select Stock Sub Group window which will come up. The group will then be inserted into the Group field.

 

Left click on the Go button to preview the report. The report will display the item code and description, when the item was last sold, the quantity on hand, and the quantity last supplied.
