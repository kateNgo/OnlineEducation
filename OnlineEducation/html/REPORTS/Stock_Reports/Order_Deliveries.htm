Order_/ Deliveries

 

This report prints a list of parts on order or which have been delivered but not finalised from the purchase order module including any parts which have been allocated to jobs.

 

On File:

 

When the Stock Orders/Delivered Report Options window appears, left click on the Include window and choose from All (a report for all items), Stock (a report for only stock items), Non Stock (a report for only non-stock items), consignment stock, revenue or trade-in. Refer to the Stock Type Topic for a full description of these items.

 

To view a list of orders or deliveries for a particular supplier/s, click in the button next to the Supplier field, and select the supplier from the Select Supplier window.

 

Summary:

 

In the Stock Orders Summary Options window, click on the button next to the Supplier field and select the supplier from the Select Supplier window.

 

Back Orders Only. Tick this box if you want the system to only show you parts that are on back order.

 

Click on the Go button to see a print preview of the report.
