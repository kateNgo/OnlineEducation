GETTING STARTED

 

Congratulations on the purchase of your AM-WIN software licence. It is with great pleasure that we welcome you to the team and wish you many years of happy computing with us.

 

We assume that you have installed your software and are ready to start using it. If this is true then it is important that you fully read this section of the manual to get off on the right foot.

 

Firstly, it is important to understand the general way in which the software functions. The keys to move around the system are the standard keys that are used by most windows applications such as the TAB key, ENTER key etc. If you are not sure about the use of the standard windows keys then you may wish at this point to read the section called Windows Functions contained in this help file.
