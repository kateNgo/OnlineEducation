Software Defaults

 

SETTING UP YOUR SOFTWARE DEFAULTS

 

Firstly you will need to set up or check your GST setting for the CASH or ACCRUAL system and the MONTHLY or QUARTERLY GST period. This is done under the OTHERS > CONTROL SYSTEM > SYSTEM & COMPANY DETAILS  section from within the main program.

 

You should now use the Control File Settings section of the help to assist you in setting up your system defaults.

 

In this section you can set up your last Invoice number, Job number, Quote number, Order number, Transaction number, Supplier number, etc.

 

You should also set up your default payment methods. You can see how this is done from the Payment Methods section of the help.

 

You now need to ensure that your general ledger accounts are set correctly. You need to make sure that your GST TYPES are correctly set for each ledger account. This is set up in the Ledger Masterfiles section. For a general overview or explanation of these accounts you should look at the section on General Ledger Setup.

 

NOTE: Journal Entries do not calculate GST. When using the software it is important to note that if you want to calculate GST then DO NOT use journal entries. A spend money entry or other type of entry must be used in this case.
