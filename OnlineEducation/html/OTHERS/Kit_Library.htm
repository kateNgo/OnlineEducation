Kit Library

 

KIT LIBRARY

 

Library is used to insert, change and delete Kits. It can also be used to reprice kits, and to add detail to kits such as parts, labour, revenue and detail.

 

Go to the Other menu and choose Kit Library.

 

Insert a Kit

 

To Insert a kit, left click the Insert button. The Inserting a Kit window will appear:

 

Type in the Kit Code and Description, and then left click the OK button, to save and insert the kit.

Please note: the Amount is the total value of the kit.

 

Changing a Kit

 

Highlight the kit in the Kit Code column, and either double left click on it, or left click the Change button. The Changing a Kit window will appear.

 

You may type in a different description, however you may not change the Kit Code or Amount. Left click the OK button to save the changes.

 

Deleting a Kit

 

To Delete a kit, highlight the kit In the Kit Code column, and left click the Delete button.

Please note: once you select that you want to delete this record, a window will appear where you must type a note on why this Kit is being deleted. This note will be stored In the Log File on Delete.

 

Repricing Kits

Repricing a kit adjusts the prices to the current prices in your AM-WIN system.

 

To reprice a particular kit, highlight the kit and left click the Reprice button. The Repricing window appears.

 

Left click on the arrow next to the Reprice Option field, and choose Selected Kit. A new Kit Code field will be inserted where the kit code for the kit you originally highlighted will be displayed:

 

If you wish to select a different kit, left click on the button next to the Kit Code field. The Select Kit window will appear, from where you may select a kit.

 

In the Reprice Kits window, you may choose whether or not you wish to reprice the stock, labour and revenue. A tick indicates that these will be repriced. You may also select to not show the error by ticking the DonShow Error box.

 

Left click the OK button. You may then choose another kit to reprice.

 

To reprice all kits, select All Kits from the Reprice Option field.

 

Left click the OK button.

 

To Add Parts, Revenue, Labour and/or Detail to a Kit

 

Highlight the kit, and then left click the Detail.

 

The Kit window for the kit you have selected will appear.

 

To change or delete any of the above information, highlight the line of information by left clicking on it, and then left click the Change or Delete button. If the Change button is clicked on, then a new window will appear displaying the information you want to change. You may then type in the new information.

To add in Labour, Parts, Revenue or Detail, left click on the Detail, Parts, Lab or Rev button. See steps 24 Â 71 in the Job Invoicing section of the manual for how to add these in, and how to use the windows which appear for each of these buttons.

 

The Total button will add in a sub-total into the kit window- ie a total for all the amounts above it. See step 76 and 77 in the Job Invoicing section of the manual for more information on this button.

 

Left click the Save button to save any changes made to the kit.

 

Left click the Close button to close the window.

 

Printing a Kit

 

To print a kit, highlight the kit in the Kits window, and left click the Print button. The print job window displaying the kit code will appear.

 

Select what you want to print and the number of copies, and then click on the Go button. A print preview will be displayed.

PRICE / QUOTE BUTTON

 

This option will provide a quote according to the customer or price or discount rate selected.

 

Select the kit you wish to price/quote for a customer by highlighting it.

 

Click on the Price button located at the bottom right hand corner.

 

The Print Job window will appear.

 

Click on the button next to the Customer field and select a customer from the list which will appear.

 

Click on the down arrow next to the Customer Price field. A menu should pop down from which you can choose which price to use.

 

Select an option from this list by highlighting it.

 

Use the up and down arrows next to the Disc Rate (discount rate for parts) and Disc Rate- Labour fields to choose any discounts you want for this customer for parts and/or labour.

Click on the Go button to see the pricing/quote report for that kit.
