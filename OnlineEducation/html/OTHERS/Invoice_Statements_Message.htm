Invoice / Statements Message

 

This section allows you to setup invoice and statement messages that appear at the bottom of your invoices and statements. You can have up to 99 different messages such as Christmas and Easter and new year messages or monthly special messages. You can then change what message is displayed on your invoice currently from the Control System section.

 

Left click on the Inv/Stat tab to add/change text footer lines to appear at the bottom of invoices and statements.

 

The Inv/Stat window will appear.

 

In the first three lines, type in the message you want to appear at the bottom of each invoice. In the last line (the 4th line) type in the message you want to appear at the bottom of each statement.

 

You may enter up to 99 different footer lines.

 

To Insert a new pair of invoice/statement footer lines, left click on the arrows next to the Text String # field and choose the next number available. Then type in the new footer lines.

 

To change a footer line, find the Text String you want to change and then type in the changes.

 

Left click the Save button after entering or changing a text footer line to save it.

 

In each Text String number you may, if you wish, choose to only fill in either the Invoice Lines or the Statement Lines.

 

The number you enter in the Use Invoice String # field in the Options 1 tab, will relate to the number in the Text String # field.

 

User Definable in Invoice and Workshop Job Description.

 

The text that is entered in this field is used to set the Standard Job Description for Job and Workshop invoicing, If this field is blank then the description defaults to General Work. This description must be set for each computer in the network and will default to the settings for that computer.
