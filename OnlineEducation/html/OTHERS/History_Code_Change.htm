History Code Change

 

HISTORY CODE CHANGE

 

This option is used to change the REGO number of a vehicle of one of your Customers.

 

Changing a REGO number

 

To change a Rego number:

 

In the AM-WIN window, choose Others from the menu, left click and choose History Code Change, left click.

 

The History Code Change window appears.

 

Enter the Old Rego number that you wish to change in the Old REG Num field. Proceed to step 10.

 

If you are unsure of the Rego number, left click on the button to the right of the Old REG Num field.

 

The Browse the Job Item file will appear.

 

To search by Job Item number, left click anywhere in the Job Item list and then type in a word or part of a word that is related to the Job Item number.

 

Note: If nothing happens when you try and search, or if the window keeps closing or selecting when you type the letter e or s: This is because the mouse cursor is not in the list. Left click in the list and type the word again.

 

This will highlight the Job Item number, or the closest match to your search. Press Enter on the keyboard, or press the Select button on the bottom of the Browse the Job Item File window, or double left click on the selected vehicle.

 

This will return you to the History Code Change screen, where the vehicle number you selected will be displayed.

 

Alternatively, to search by Customer Name, hit the tab titled By Customer at the top of the Browse the Job Item File window. You can then search for a job item by customer, group, make, model and 2 other user definable fields.

 

This will bring up the list of vehicles, which that customer owns

.

Left click on the vehicle whose REG number you wish to change, and press Enter on the keyboard, or the Select button, or double left click on the selection. This will return you to the History Code Change screen.

 

Once the old REG number which you wish to change is displayed in the Old REG Num field, press TAB on the keyboard or place the mouse cursor in the New REG num field and left click.

 

Type in the new Reg number which you wish to change the old rego number to, and press the Process button located at the bottom of the History Code Change window.

 

This will update the Job Item files and the service history.

 

Once the update button has been pressed, the History Code change window will be cleared in order for you to enter another REG number to be changed.

 

If you have completed your changes, left click the Close button at the bottom of the History Code change window, or left click the X at the top right hand corner.
