Letter Maintenance

 

LETTER MAINTENANCE

 

Letter maintenance is used to keep a record of letters sent out to customers and/or suppliers. You may insert a letter as well as update a previous letter.

 

 

Go to the Others menu and choose Letter Maintenance. The Browse the Letter file window will appear.

 

You may search for the Letter you want by the Name of the Letter, in the By Name tab.

 

If you left click on the By Code tab you can search for the letter you want by the Letter code.

Note: even though the names of the letters (and not the codes) appear in the list in the By Code tab, the names are arranged in order of the codes for that letter.

 

To Insert a letter, left click the Insert button. The following window will appear.

 

Type in Letter Code, Name and Description in the Code, Name and Desc fields. Type the Letter into the large Letter field. Left click the Ok button when finished.

 

To change a letter, highlight the letter name on the list and left click the change button. The Changing a Letter Record window (similar to the inserting a letter record window) will appear showing the current details of the letter you want to change. Make the changes you want and then left click the OK button.

 

The Delete Button will allow you to delete a letter.

 

Left click the Close button to close Letter Maintenance.

 

By Item

 

This button will allow you to print service due letters for all job items due for service. You can then select the specific Item or the type of letter than you wish to send.
