Passwords

 

Left click on the Passwords tab on the top of the Control System menu in order to enter and/or change an operator, their password and their password level. The Passwords Controls window will appear.

 

To Insert an operator, left click the Insert button.

 

Type in the Operators initials in the Operator Field.

 

Type in their password in the Password field.

 

Type in their password level, OR use the arrows to the right of the level field to choose what password level that operator will have.

 

Hit the Enter key, or click on the Update button, and that operator will be added to the list.

 

To Change an operators password or level, left click on that operator in the list. Type in the new password and level into the password and level fields, and press the Update button. You may not change the Operator name. To do this you must delete the operator (by left clicking the Delete button) and re-enter the operator with a different name.

 

P_Levels 1

 

Left click on the P_Levels 1 tab in the Control System window. This is where you may set or change the password levels for the Customers, Ledger, Stock, Reports, Others and Suppliers sections in AM-WIN.

The levels set here are the minimum password levels an operator must have to access these sections. The P_Levels 1 window appears.

 

AM-WIN will have automatically inserted password levels into each field. To Change any of these, left click in the field for the section you want to change, and type in a new password level.

 

Left click the Save button to save these changes.

 

P_Levels 2

 

Left click the P_Levels 2 to set and change the password levels for Accounting Reports, Stock Reports, Workshop Reports, Customer/Supplier reports, Employee and others. These levels are the minimum password level an operator must have to access these sections. The P_Levels 2 window appears.

 

AM-WIN will automatically insert password levels into these fields. To change a password level, left click into the field you want to change, and type in the new level. Then click on the Save button to save the changes.
