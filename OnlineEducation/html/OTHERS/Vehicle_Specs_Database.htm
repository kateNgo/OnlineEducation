Vehicle Specs Database

 

VEHICLE SPECS DATABASE

 

The Vehicle Specs Database holds data on specific vehicles/job items.

 

You first need to set the headings you want to be used in the Vehicle Specs Database:

 

MODIFY Job Item SPECS FIELD LABELS

 

Go to the Tools menu and choose Modify Job Item-Specs Field labels. The window Fields Labels window appears.

 

Simply click into the Different Fields and delete the blue Heading words in that field. Then type in the Heading you want.

 

Once you have completed filling in the Headings, click on the Save button.

 

You are now able to use the Vehicle Specs Database

 

VEHICLE SPECS DATABASE

 

Go to the Others menu and choose Vehicle Spec Database

The Browse the Vehicle Spec Database window will open.

 

Click the Insert button or hit the Insert key on your keyboard. This should put this window into Insert Mode.

 

Type in the Spec Code. Then click into each of the other fields, filling in information for the particular job item/vehicle. You may click on the Notes tab at the top left hand side of the window in order to type any notes on the vehicle.

 

Click the Save button when you have completed filling in the information. The record will then be added to the left hand column.

 

To Delete a record in the Vehicle Specs Database, simply select this record in the left hand column so that it is highlighted, and then click on the Delete key.
