Control System

 

CONTROL SYSTEM

 

!! Warning !!  The control system is a maximum security section of the software and incorrect settings may cause the system to function incorrectly.

 

The control system is used to enter and change your Business details, password levels and passwords, the last invoice, job, quote and order numbers, text appearing on your invoices, printer selection, and various other information which deals with your AM-WIN system.

 

Go to the Others menu and choose Control System. The control files window will appear.

 

Business Tab

 

In the Business tab, you may change any of the Company Details, except for the Company Name. You must contact AM-Win Software or your agent in order to change the company name on your system.

 

To enter in or change any of the Company Details, left click in the field and type in the information you require.

 

Printer Selection

 

AM-WIN will allow you to have up to four different printers. In the Printer Selection fields the Default Printer is automatically inserted into the first field. This cannot be changed and is used as the report printer. Next to the default printer tick the Have laser Printer box if you have a laser printer installed. This determines the way in which the software handles the printing.

 

The Labels printer can be chosen by left clicking on the button to the right of that field. The Print Setup window will appear.

 

Choose the printer you want by left clicking on the down arrow in the Specific Printer field and choosing from the menu which pops down. You may choose to use the default printer for the Label printer, in which case, left click in the small circle next to Default Printer. Left click on the OK button, and the printer you selected will appear in the printer selections fields.

 

The Job card, Quotation and Invoice/Statement printers can be selected as per the label printer above.

 

Invoice Format

 

In the Others menu within AM-Win under the control file option through the options TAB you can select 5 different invoice formats to suit your particular taste. Try these out to see which one you like the best and which one best suits your existing stationary. Some of these formats have been designed to allow you to use your existing stationery when coming to AM-Win from a competitive product.

 

Invoice String

 

In the Inv/Stat tab of the Control System you may enter in text footer lines to be printed on invoices and statements. You may enter up to 99 of these. The number you select to be in the Invoice String # field, will indicate which Text Line number you wish to use at the moment. For example, if the 25th text footer line you added in was a Christmas message, you may put in 25 in the Use Invoice String # field during Christmas ( refer to the Inv/Stat tab for better understanding).

 

Copies

 

Within the Control System there is an option where you can set the number of invoices you want to print. Therefore, if you always want to print 2 copies of an invoice, you can set 2 as the default, so that each time you print an invoice you donhave to manually change the number to 2 each time.

If you have set a number as a default, you can always change this number in the Print Options screen at the time of printing a particular customer

 

Click onto the arrows next to this option to set your default number of copies. Then click onto the Save button at the bottom of the Control System.

 

NOTE: This function will only work on Laser printers and a few bubblejet printers.

 

Offset From Top and From Bottom

 

Many users of the software have letterhead paper that they wish to use for printing invoices and statements etc. Because not all printers start at the same place on the page such as lasers and bubble jets which start at different places, we have included the option to start the page printing in a lower position and also to finish the printing at a higher position. To set the page position, under the Others menu within AM-Win and then under the control system option and then under the Business Tab you are able to change the Offset from top and Offset from bottom settings. 100 units is equal to an offset of approximately 2.54 mm or 1/10th of an INCH in the imperial scale. This option will only work for the invoice, credit returns, statements printer setting.

 

Options 1 Tab

 

Left click on the Options 1 tab at the top of the control system window or press the Ctrl  Tab keys together, and the options 1 tab window will be displayed.

 

 

System Options

 

The system options are not user definable and show the modules that are licenced for this user.

 

Country

 

This indicates the country that the licence was issued for.

 

Stock Catalog Controls

 

Access to Stock Repricing and Export Functions:

 

Determines whether or not to allow access to the functions in the AM-WIN catalog that allow parts to be exported or repriced to the AM-WIN stock file.

 

Display Cost Price:

 

Determines whether or not to display the cost price of the stock that is held in the AM-WIN Catalog files.

 

Replace Groups:

 

Determines whether or not to replace your existing stock groups with the groups that are supplied on your suppliers parts catalogue disk when repricing parts that are in the AM-WIN stock file.

 

Replace Stock Desc:

 

Determines whether or not to replace your existing stock descriptions with the descriptions that are supplied on your suppliers parts catalogue disk when repricing parts that are in the AM-WIN stock file.

 

Last Assignments

 

These group of numbers indicate the last number assigned to each type of transaction such as the last invoice number, job number etc. These can be increased but not decreased so that duplicate numbers are not allowed. If this number is incorrectly set you will get a Duplicate Key error which mean the numbers need to be increased. Increasing the numbers by 1 will usually repair the error.

 

Invoice # - This indicates the last customer invoice number.

 

Job # - This indicates the last customer job number.

 

Quote # - This indicates the last customer Quote number.

 

Order # - This indicates the last supplier order number.

 

Trans # - This indicates the last ledger transaction number.

 

Supplier # - This indicates the last supplier transaction number.

 

Invoice Options

 

Show GST on line item in Invoice

 

You can now set your invoices so that each time you print an invoice the GST appears for each item on your invoice, on each line. Therefore when the Print Option window appears after you have finalised an invoice the Show GST on Line Item tick box will be automatically ticked.

 

Go into the Control System and click onto the Options tab. Under Options 1 on the left hand side there is a tick box called  Show GST on line items in Invoice.

 

Right click into this tick box so that it shows a tick in it. Then click onto the SAVE button at the bottom of the screen.

 

Print Business details on Invoices/Statements/ Credits/Letters

 

The company details in your Control System will be printed on these documents if the box is ticked. If you untick the box then they will not be printed and this will allow you to use pre-printed stationary.

 

Print Line Amount

 

If this box is ticked, then the individual prices for each stock item will be printed on the invoice. If it is not ticked, only the quantity of each stock will be printed on the invoice, as well as an overall amount for all the stock.

 

Print Labour Hours

 

If this box is ticked, then the quantity (number of hours) of labour will appear on the invoice, as well as the amount for that labour. If it is not ticked, then a Labour amount will appear on the invoice, but not a labour quantity.

 

Combine Labour Hours 

 

If this box is ticked, then the invoice will appear with only one total labour amount at the bottom of the invoice. If this is not ticked, then a separate amount will appear for each set of labour.

 

Print Stock Codes

 

If this box is ticked then the stock code will appear next to the stock item on the invoice, otherwise the stock part number will not be printed at all.

 

Display Invoice/Quotes etc Including GST

 

If this box is ticked, then the line total amount will include GST. It is important to note that in order to see the line totals on an invoice, you must de-select the DonPrint Line Amounts option on the print job window which comes up when a print button is pressed, so that no tick appears in that box.

 

Re-price Stock from Job Invoice

 

If ticked then this option allows the price of a stock item to be changed from Job invoicing by clicking on the re-price button when entering a part onto an invoice. More details are available in the Insert Parts help.

 

Must enter salesperson code on Invoice

 

If ticked then an invoice cannot be saved or finalised unless a salesperson has been selected for that invoice.

 

Decimal place in Invoice unit price

 

From the Others menu in AM-Win under the control system option and the options 1 tab you can set the number of decimal points to be printed on invoices and purchase orders. This is default to 2 decimal places but you may wish to change this if you are selling petrol or other items that calculate at more than 2 decimal places.

 

Progressive Job Invoicing

 

The system allows you to finalise only part of an invoice at a time if this is what you wish. This is known as Progressive Invoicing. This allows you to invoice the same job to different customers (insurance and warranty work) or the same job several times to the same customer (say the end of each month).

 

To access this feature you need to go to the Others menu>Control System>Options 1 tab. Right click into this field so that a tick appears, and then press the SAVE button.

 

If you want a Customer to always have Progressive Job Invoicing, go into Customer Masterfiles, for that Customer and click onto the Customer Accounts tab. On the right hand side click into the field called Use Progressive Invoice.

 

Once this is clicked it means that each time you do an invoice for this customer, you are able to finalise it progressively. Therefore in the job invoicing screen, on the top left hand corner the below field will be ticked.

 

Must Enter Meter Reading in invoice

 

If this option is ticked then you will not be able to finalise a job invoice unless you enter a meter reading against the job item.

 

Ignore Print Preview Window

 

If you place a tick in the box then from now on, when you click on the Print button within invoicing, quotes, purchases and credits, the print options window will appear. Once the Go button is clicked on this print option window, the document will print automatically, without showing you a print preview screen first.

 

Round Invoices to Nearest 5 cents / $1

 

If this is set to Yes, then prices and amounts which appear on invoices will be rounded to the nearest 5 cents. If this is set to No then the exact amounts will appear on invoices. If this is set to $1 then prices and amounts which appear on invoices will be rounded to the nearest $1.

 

Print Job Card Authorisation

 

There is an option in the Control system for you to choose the Authority text to always or never print on job cards

 

Right click into the tick box which says Print Job Card Authorisation so that a tick appears if you want the authority text to always print on job cards. Leave this box blank if you donwant this text to print on job cards.

 

Donuse bill to in Job Cards

 

Usually when an invoice has a Bill To which is different to the Customer, the Bill To details shows up on the Job card.

 

However, if you go to the Others menu, and then to the Control System, and then into the Options 1 tab you will see a tick box option half way down the page under User Options which says  Donuse Bill to in Job Cards

 

If you want the Customer details to show up on job cards which have a different Bill To customer, then right click into this tick box so that a Tick appears. Your job cards will now automatically always show the Customer details, and not the Bill To Details

 

Print Memo on Job Card

 

If this option is ticked then the Job Item notes (if any) will be printed at the bottom of the job card for the selected job item.

 

No of Ruled lines to add in job card

 

Use the up and down arrows next to the No. of ruled lines to add in Job Card box to set the default number of ruled lines which are printed on a job card.

 

GST / BAS settings

 

Current Month

 

Warning !!

 

This displays the current working month and should never be changed unless advised to do so by an authorised AM-Win distributor or agent. Changing this field may damage your data.

 

GST Method

 

This field sets the GST method such as Cash or Accrual to be used for processing your GST records. Please ensure this is set correctly. Contact your accountant if you are unsure.

 

BAS Period

 

This field sets the BAS period such as Monthly or Quarterly to be used for processing your BAS records. Please ensure this is set correctly. Contact your accountant if you are unsure.

 

GST Tax Rate

 

This displays the tax rate of the GST. This rate may be changed (when/if the Government changes the GST rate) by typing the new rate into the field.

 

Financial Year End

 

This field is used to set the date of the next Financial Year End, and may be changed by typing into it. It is automatically updated at the end of each financial year.

 

Warning !!

 

If this date is incorrectly set then your accounting system will not function correctly and will report figures for the wrong financial year.

 

Options 2 Tab

 

ASM Options

 

The ASM (Auto Service Manager) button accesses the ASM program which is a third part program the software links to. ASM contains the service details for over 500 makes and models of vehicle including the service book parts, labour and service requirements for each regular service for each vehicle.

 

Activate ASM

 

If this box is ticked then the ASM option will become available for use in the JOB Invoicing and Workshop invoicing Module.

 

ASM Program Path

 

This is the path that that ASM program is located in. Click on the button next to the path to set the default standard path or manually type in the required path.

 

User Options

 

Hide the main menu Tool Bar

 

Many of the users of AM-WIN prefer to use the menu options at the top of the menu screen rather than using the 18 shortcut icons across the screen just below the menu options. If you are one of those people then you are able to hide the shortcut tool bar either temporarily or permanently. You can hide it temporarily by selecting the file option from the menu screen and then select the hide/unhide tool bar option. To unhide the tool bar simply select the File > Hide/Unhide tool bar option and the tool bar icons will again be displayed.

 

If you wish to permanently hide or unhide the tool bar icons you can do so from the Others > Control file option. You will need a Password level of 99 to access this option from the control file if you have password security activated. Once in the control file screen you need to click on the Options TAB and click the hide tool bar option on the right side of the options screen. You can unhide the tool bar by un-ticking this option.

 

Under the Others menu within AM-Win and then under the control system option and then under the Controls 1 Tab you are able to Hide the tool bar which contains the 18 most commonly used icons. See Section 7 of this manual for further details.

 

Workshop Diary on Opening AM-Win

 

If this box is ticked then the software will automatically open the Workshop Diary when starting AM-Win. This option is defined separately for each computer and therefore in a multi-user environment only computers where this option set will open automatically into the diary.

 

Default to Name search for Ledgers

 

Usually when searching for a ledger account you have an option to search by Code or by Name, with the By Code option being first. However if you tick the field  Default to name search for ledgers when you search for a ledger account the first screen will be the By Name search field as a default.

 

Do not show balance owing on Remittance

 

Under the Others menu within AM-Win and then under the control system option and then under the Options 2 Tab you are able to select to not show the customers or suppliers outstanding balance on the remittance advice forms.

 

Do not show employee details on Opening Window in Workshop Diary

 

If this box is ticked then the window showing the employee hours will not be open automatically when the workshop diary is opened.

 

Update Preferred Supplier from Purchase

 

If this is ticked, then the preferred supplier on the stock file will be updated when a purchase is made for that stock item.

 

Activate Password Security System

 

If this is NOT ticked, then the passwords and password levels you set in the Passwords, P_Levels 1 and P_Levels 2 sections will NOT be used and any operator will be able to access any section within AM-WIN.

 

Allow Job Item Stock Search

 

This option allows you to link parts to a selected job item so that when you work on that particular job item (vehicle, boat, lawn mower etc) you have a list of specific parts that fit that item.

 

Turning on the linking option

 

Firstly you must go into the Others menu of AM-Win and then into the control system option and then to the options 2 tab. If this box is ticked then the option is turned on.

 

Customer must have Group

 

If this box is ticked then a when adding or saving a customer, the customer must be allocated a group.

 

Use account keeping fees.

 

Account keepings fees are a separate option under the Ledger/Month end option menu. Account keeping fees can be processed for a particular amount or a percentage of the customers outstanding balance. Please note that this should only be run once! If this box is ticked then you will get a reminder message to ensure that you have run the account keeping fees process before closing the month.

 

Activate PAYG.

 

PAYG (group tax) can be paid monthly or quarterly regardless of your BAS period settings. You must select here the PAYG period as being monthly or quarterly.

 

Report Percentage Cost on Revenue

 

This allows the user to default a percentage figure used to calculate the cost price of a revenue item from the pricing window in Job Invoices and Quotes using the [C] button option to display the cost price. Etc. If a revenue item is charged at $100 and the Report Percentage Cost is 80% then the cost of the $100 part will be shown as $80.00. This is an estimate only and is not used to calculate your profit/loss or GST figures. If a COST has been entered into the GL code for the revenue item then this will be used in preference to the Report Percentage Cost.

 

Report Percentage Cost on Labour

 

This allows the user to default a percentage figure used to calculate the cost price of a labour item from the pricing window in Job Invoices and Quotes using the [C] button option to display the cost price. Etc. If a Labour item is charged at $100 and the Report Percentage Cost is 80% then the cost of the $100 part will be shown as $80.00. This is an estimate only and is not used to calculate your profit/loss or GST figures. If a COST has been entered into the GL code for the revenue item then this will be used in preference to the Report Percentage Cost.

 

Ignore stock level of zero  Intelli Search.

 

In the Intelli Search for stock there is an option to Ignore zero on hand quantity. If the field Ignore Stock Level Zero-Intelli Search in the Control System is ticked, then the Ignore Zero on quantity field will automatically be activated in the Intelli Search, and the search will not pick up stock items which have no stock on hand.

 

Default Stock Price Display on Lookup

 

Indicates which price you would like to have displayed in the price window when you are looking up a stock item.

 

Activate Minimum Stock Profit Allowed

 

If a minimum stock percentage is entered then AM-WIN will warn you if a stock item is sold for a profit margin of less than the minimum margin entered here.

 

Left click the Save button to save any changes.
