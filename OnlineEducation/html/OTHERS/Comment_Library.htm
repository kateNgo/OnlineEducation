COMMENT LIBRARY

 

Use this option to add comments to use for your invoices.

 

In the AM-WIN window, choose Others from the menu, left click, and choose Comment Library, left click.

 

The Browse the Edit Text window will appear.

 

To INSERT a comment, left click on the Insert button then type in the text name..

 

Left click on the Name field and enter in a name for the comment

 

Press the TAB key or left click on the large Text detail field and type in the comment.

 

When you reach the end of a line in the comment field, it will not let you type anymore. At this point press the TAB key to continue typing on the next line.

 

Click the OK button to insert this new comment into the comment list and return to the Browse the Comment Library Window.

 

To DELETE a comment, left click on the comment you wish to delete in the commentname list.

 

Left click on the Delete button.

 

A message will pop up asking whether you are sure you wish to delete this record. Left click on the YES or NO button depending on what you wish to do.

 

To CHANGE a comment, simply double left click on the comment you wish to change, or, left click on the comment you wish to change once, and then left click on the Change button.

 

Once you have made the changes, left click on the OK button. This will change the comment and return you to the Browse the Comment Library window.

 

To SEARCH for a comment, left click anywhere in the Text Code, and type a word or part of a word which is related to the comment name you want. This will highlight the desired comment name or highlight the closest match.

 

When you completed using the Comment Library, left click the Close button or the X button on the top right hand corner.
