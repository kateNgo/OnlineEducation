Payment Methods

PAYMENT METHODS

 

Choose Method from the Maintenance. The Method File window will appear.

 

To Insert a method, left click the Insert button, and the Adding A Method File Record window will appear.

 

Type in the Method you are inserting, eg- Amex, Visa etc

 

Left click the button next to the Met Code to choose the General Ledger code for the method you have chosen.

 

Enter the Type including B  Bank/Cheque, C- Cash, D  Direct Deposit, E  Eftpos, X  Credit Card

 

Left click in the small box next to Permission to Delete if you want it to be possible for this method to be deleted.

 

Left click the OK button when you have filled in the above details, and the method you have added will be inserted into the Method File window.

 

To Delete a method, highlight the method in the Method File window, and left click the Delete button.

 

To Change a methoddetails, highlight the method in the Method File window and left click the Change button. The changing Method File Record window will appear.

 

You may change the Met Code, but you may not change the Method name, or the Permission to Delete. Then press the OK button and the changes will be saved.
