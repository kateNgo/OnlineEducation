BROWSING THE POSTCODE FILE

Left click the button on the right of the postcode field or from the menu choose Others > Group Maintenance > Postcodes.

The Browse the Postcode File window appears.

Choose Insert to create or add a postcode to the list.

Left click in each of the fields and enter in the suburb, state and postcode and select OK to record the postcode details.

In the Browse the Postcode Window, Left click on the applicable Street, Suburb, State, Postcode and this highlights the details to be automatically inserted in the address field, then choose Select. Choose Change to modify the highlighted postcode details, or choose Delete to remove the highlighted postcode details from the postcode file.

Note: you should no longer need to enter the postcode/suburb manually once the Postcode File is set up.
