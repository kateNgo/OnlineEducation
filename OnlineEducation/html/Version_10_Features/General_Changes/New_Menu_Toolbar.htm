New Menu Toolbar

AM-Win has introduced into the software a new menu toolbar containing fresh and updated icons. The toolbar is no longer pinned to the top of the screen and can be moved to either the left or right side, or to the bottom of the screen.

By left-clicking and holding the mouse button down on an empty area of the toolbar, you can drag the toolbar using the mouse to a new position on any edge of the screen.

A few additional icons have been added to the toolbar, including a new ‘Reports Centre’ icon.
