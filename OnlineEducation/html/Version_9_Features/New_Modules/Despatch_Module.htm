Despatch Module

 

The Despatch module is designed to be run at the warehouse despatch area simply for the despatch of parts from the warehouse to the customer, usually via a courier. This module allows you to print picking slips, create a despatch/deliver docket, print despatch labels, organise courier runs and nominate the qty of items and courier for the despatch.

 

The despatch screen looks similar to the Find Customer order screen. However, the despatch screen contains all of the required functions and you are not able to actually open and change a customer order other than to nominate the despatch quantities and courier details.

 

You can filter the list of outstanding customer orders from the filter options at the top of the screen to display a smaller number of orders or a specific order. You can then update the order despatch quantity by clicking on the order detail lines and updating the despatch quantity at the bottom right of the screen and then clicking on the update button or simply pressing the enter key.

 

You can then select the courier if one has not already been assigned and fill in the appropriate details such as the date, pickup time, number of cartons, consignment number etc. You can also assign a user definable status to the order.

 

You can print a picking slip to pick the parts using the picking slip button and also print the despatch mailing labels for the boxes. The system will print a label for each carton with the delivery details and courier / consignment details on the label.

 

You can then use the despatch button to despatch the goods, update the despatch date and print the delivery docket to be sent with the goods.
