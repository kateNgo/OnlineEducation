Security & Passwords

 

To activate and use the security and the pass wording feature of AMWIN you must go through a set up process. Firstly you must set up the operators (ie all users who will need to login to use AMWIN). To do this you go to the Others>Control System>Operators menu. The Users & Password settings screen will appear.

 

From this screen click on the Insert button to enter in the Operator initials, the unique password and level you wish to assign to this particular employee. If you have Multi location you must choose a location they are assigned to. You also have the option to choose a purchase and invoice authorisation level for that operator. Once you have entered in all the relevant fields, click on the Save button and close out.

 

The second thing you need to set up would be the different security levels allocated to each section of the AM-WIN Program. Go to the menu Others>Control System>Security Control. The Security Control/Business Options screen appears.

 

From this screen you need to fine tune the different security levels for the different sections to be in line with the security level you have allocated to different operators. Once you have made your adjustments, click on the Save and Close Button.

 

The Final thing you must set up before Pass wording and security is activated is to turn the security on. To do this go to Others>Control System>System & Company Details, click on the Security/Options Tab, then Under the Employee & Security heading, tick the box to Activate Password Security System.

 

Once you tick and save this option activation, close out of AM-WIN and open it back up and the login prompt will have changed, asking you to log in with your username and password. From this point on, any transaction completed in AM-WIN will be traced by Operator.
