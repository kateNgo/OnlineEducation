Stock Data Tools

 

Stock Data Tools

 

The AM-Win Stock Data Tools module is a power tool which allows you to manipulate your stock file in a variety of ways. From setting prices, changing descriptions, moving parts between and in and out of stock groups and sub groups, changing preferred suppliers to many other features. This tool provides a great deal of power for you to make your stock file much more efficient and for it to start working for you rather than against you.

 

The Stock Data Tool is not a standard AM-Win module and can be purchased through your local AM-Win distributor or agent.

 

Rules

There are several rules and limitations associated with the Stock Data Tool that must be understood in order to obtain the most benefits from it and also to ensure that you ask it to do things that adhere to both the accounting and general data integrity of AM-Win.

 

Firstly, there are a number of changes that can only be made when the stock on hand quantity is zero. In this case, if some of the stock in your batch is zero on hand and others have a quantity on hand other than zero then only the zero on hand items will be changed when you process your request. Such limitations are as follows:

 

  1. You cannot change the stock class of a stock item whilst there is stock on hand. This is because Stock items carry a value in the ledger whereas non-stock and revenue items don. If the system allowed these changes to be made then the accounting integrity of the stock ledgers would be adversely effected. The stock value in the ledger would not represent the value of stock in the stock file.

 

  1. Similarly to limitation 1, you cannot change the quantity on hand for a stock item but you can for a non stock item.

 

  1. You are not able to change the GST flag from GST to no GST or vice versa while you have stock on hand.

 

  1. You are not able to change the is serialised option for stock when there is stock on hand.

 

  1. You are not able to change the Asset GL Code for stock when there is stock on hand.

 

  1. You are not allowed to set the recommended retail price of a stock item below price 1. (Mainly just to eliminate silly mistakes)

 

  1. You are not able to change the creation date of a stock item if the creation date already exists (For users on the software prior to version 6)

 

To Create a New Batch or to Revise an Existing Batch

When you enter the Stock Data Tool screen you have the option to either create a New batch or to revise an existing batch. To create a New batch you must first click on the New button at the top left of the screen.

 

New Button - When the New button is selected the next available batch number is automatically assigned. You should then enter the Batch Description which can later be used to assist you in revising the correct batch for the job you want to do. The description is required before you can save or process a batch.

 

Stock Class - Allows you to select from all items, Stock items, Non-Stock items, and revenue, trade-in or consignment stock items. Using this filter you can determine what stock class you wish to create the batch from.

 

Sort By - Allows you to select the default sort criteria for the items to be placed in the batch list area. You can sort the list by Stock Code, Bin location, Group, Sub Group, Bin / Stock code, Description or preferred supplier.

 

Now you have set the Stock Class and Default sort criteria you can choose to either generate your stock batch using the Single, Multi or Orphan tab options.

 

The Single Filter Tab

The Single Filter tab allows you to use a variety of record filters to generate your batch of parts.

 

Filter by All Items

All Items  This option allows you to select a range of stock by part number for all items available within the selected stock class you have chosen. Once you have selected the ALL option you then must choose the From Part item. If you wish to select the entire list of items then simply select from the first item and to the last item. This will create a batch containing every item in your stock file that matches the selected stock class.

 

NOTE: Some changes to such a potentially large batch may take considerable time to process so you may be better served to process several smaller batches rather than a single huge batch.

 

From Part - Click on the From part button and select the item to begin the batch with. You are able to use the usual locator and search options to find the item required.

 

To Part - Click on the To part button and select the item to finish the batch with. You are able to use the usual locator and search options to find the item required.

 

Once the From and To parts are selected you can then click on the Generate Button to bring that list of items into the batch. Those items are then placed into the list box to be worked on. You can then either add further to the batch by using the Single Filter Tab or you can set your Data Change on the right of the screen.

 

To Filter by BIN Location

The by Bin Location filter allows you to generate the batch between bin locations. Firstly click on the Bin Location option. You may notice a delay which can last from moments to several seconds depending on the size of your stock file. This is the time the system requires to build a list of bin locations for you to select from. Once the From and To Bin fields are available you can select tour choices.

 

From Bin - Allows you to select the bin location to begin the batch from.

To Bin - then allows you to select the bin to finish the batch at. The To Bin will only show bin locations which are alphanumerically after the From Bin.

 

Once the From and To bins are selected you can then click on the Generate Button to build the batch based on your selections. You can then either add further to the batch by using the Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

To Filter by Group

The by Group filter allows you to generate the batch between groups. Firstly click on the Group option.

 

From Group - Allows you to select the group to begin the batch from.

 

To Group - Then allows you to select the group to finish the batch at. The To Group will only show groups which are alphanumerically after the From Group.

 

Once the From and To groups are selected you can then click on the Generate button to build the batch based on your selections. You can then either add further to the batch by using other filter options Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

To Filter by Preferred Supplier

The by Supplier filter allows you to generate the batch based on the preferred supplier. Firstly click on the Supplier option.

 

Supplier Code - Click on the Supplier code button and select the supplier to be used.

 

Once the Supplier has been selected you can then click on the Generate button to build the batch based on your selection. You can then either add further to the batch by using the Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

The Multi Filter Tab

The Multi filter tab allows you to filter the required stock batch by using the Preferred Supplier, Group, and Subgroup and Bin location range in combination with each other. You are able to select any of these options on their own as well. For instance if you only select the subgroup then only the subgroup will be used to filter the item. However if you select a combination of say the group and subgroup then only items where both selections match will be generated in the batch. This means you can limit the range within the batch to parts only with a specific supplier and group and subgroup and even then only if they fall in a certain bin location range.

 

Once you have selected your range of filters, click on the Generate button to build the batch based on your selection. You can then either add further to the batch by using the Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

The Orphan Filter Tab

The orphan filter tab allows you to select a range of stock items which currently are considered by the system as orphaned. This means they either have no group, no subgroup, no preferred supplier or no Bin location or a combination of any of these.

 

This allows you to locate and adjust items that have escaped your usually allocation for them. You may wish to select all items that do not have a group and group then as NOGRP. This no group, group can then be utilised as a group of items without a group. As such they can then be used in the multipliers, discount matrixes and other group functions within AM-Win. Alternatively you may wish to choose only a section of the batch generated and then put those items into a specific group and then others to another group etc. To do this you must first delete the items from the batch that you do not want effected when you process the batch changes.

 

Tick the boxes next to the orphan selections you want included and then click on the Generate Button to build the batch based on your selections. You can then either add further to the batch by using the Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

The Operator

The operator records the name of the user who processed the batch changes. These details are stored in the log file so that stock data changes can be tracked.

 

The Batch Process Option

You have a choice to select Batch Only, or Batch and Process. The only difference is that the change Options area is not available if you choose the batch only option. This allows you to generate stock batches with the need to have the options area of the screen open making the screen less crowded and easier to operate.

 

Ignore Zero Stock Option

If the Ignore Zero stock option is ticked then when the Generate button is selected, the system will only generate the list for items with stock on hand.

 

Active Stock Option

If the Active stock option is ticked then when the Generate button is selected, the system will only generate the list for items with a stock status of Active.

 

Generate Button

Once the filter criteria have been set using the Single Filter Tab, you can then click on the Generate button which generates the batch list based on the selected filters. You can then either add further to the batch by using the Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

Records Field

The Records field displays the number of records that are currently in the batch. While there is no limit to how many records can be placed into a batch, we recommend a few hundred or so as being a manageable batch size. This of course widely varies depending on the task you wish to perform on this batch.

 

Insert Part

The Insert Part button allows you to insert parts by manually selecting the parts to be included in the batch.

 

Firstly click on the Insert Parts button. This will display a stock lookup screen whereby you are able to locate parts either by part number or by description. You can also manually scroll through and select the required parts. Once you have located the part required you can either press the Enter Key or click on the Toggle button to select the item. You can then locate and select other multiple items as well. The selected items will be highlighted in red. Once all of the required items have been highlighted you can then press the F12 key or click on the Select button to select the required items.

 

The items are then added into the stock batch. You can then either add further to the batch by using Single Filter Tab or you can set your Data Change Options on the right of the screen.

 

Delete Items

The delete button allows you to delete the highlighted line from the batch. You may have used one of the filter options to bring a range of items into the batch and then want to delete several items within that range that you do not want to change. You can also use the Ctrl-Del keys to delete the highlighted item.

 

Locator

The Locator Button allows you to search for and locate a specific stock item within the batch. You can also type the first character of the part number whilst in the batch list to locate a part starting with that first character.

 

Once the Locator button has been pressed you can search for the part number in the usual way. You are able to search by part number, description, x-ref, bin location, barcode and also by using the intelligent search function. Help on these search facilities is available within the AM-Win help.

 

Once the item required is found then you can select it and you will be taken to the selected item in the stock batch. If the selected item is not in the batch then a message stating this will be displayed.

 

Inside the Batch List

Whist highlighting records within the batch you have a variety of options available to you. The list displays the Part Number, Description, On Hand qty, Group, Bin location and Preferred Supplier. Sometimes these fields will not all fit on the screen at once. In this case you can narrow or widen the columns in the batch list by the usual windows method. Move the mouse between the columns until left and right arrows appear where the mouse cursor is usually shown. You can then hold the left mouse button down and move the mouse left or right to narrow or widen the columns. You are also able to access the columns by clicking on the horizontal scroll bar at the bottom of the batch list so that the columns move over to the left or right.

 

The screen also displays the Selling prices, Sub group, Cost prices and Preferred Supplier of the selected item at the bottom of the screen.

 

If you highlight a line within the batch you can click on the Delete button or the Ctrl-Del keys to delete the selected item. You can also sort the items into part number or Description order or reverse order by clicking on either the part number or description heading.

 

You are also able to tag and un-tag multiple items within the batch by entering on them or right clicking on the item and selecting the toggle option. Once tagged you can then right click on the batch list and delete all of the tagged items by clicking on the Delete all selected option.

 

When you right click inside the batch list you can also tag or un-tag all items within the batch list. When you process option changes you can then choose to process only tagged records or the entire batch.

 

Save Button

Once the batch has been generated, you can then save the batch by clicking on the Save button. You can then retrieve that batch at a later stage using the Revise Button.

 

If you have not selected the operator then you will be prompted to do so prior to the batch being saved.

 

Clear Button

The Clear button will clear the current batch list and reset the New and Revise buttons to active ready for the next transaction. Any unsaved changes will be lost if you clear the batch list.

Save/Print Button

 

New Mode

The Save-Print button (or Print Batch button in revise mode) will save and print the current batch. This allows you to visually see the batch on paper before processing mass update changes to the batch.

 

Revise Mode

This button is labelled Print Batch when in revise mode but effectively performs the same process as above.

 

Revise a Batch

Once a batch has been saved you can then revise the batch using the Revise Button. After clicking on the revise button a list of available batches will be displayed. You can then click on the required batch which brings that batch into the batch list. Once the batch is displayed in the list you can then add additional items into the batch using the Batch Filter options (Single Filter Tab) or you can print (Save Print Button) or even delete (Delete Batch) a batch. The batch can also be used to process batch option changes.

 

Delete a Batch

Once a batch has been saved or a saved batch has been revised you then have the option to delete the batch. Select the batch you wish to delete and it will be loaded into the screen. You can then click on the Delete Batch button to delete the displayed batch.

 

Options

Before you are able to utilise the many functions available in the Data Change options you must be aware of certain rules and limitations (Rules) required ensuring the integrity of your data. You then must have generated a New batch (New Batch) or be Revising (Revise a Batch) an existing batch which will be amended by the Stock Data Tool.

 

Now that a batch is selected you can make the following changes.

 

New Description  This option allows you to set the entire list of items in the chosen batch to have the same description. You may want to have a batch of spark plugs with different names and make them standard to read Spark Plugs. Previously they may have had names like Bosch plugs, NGK Plugs, Champion Plugs and now you just want to call them all Spark Plugs.

 

GST  The GST option allows you to change the status of the GST for selected items. Note that the GST change will only affect parts in the list which has a zero stock on hand value. The cost ex and Inc GST fields will also be made the same with the Inc GST field being made equal to the current ex GST price. If left blank then the option is not changed.

 

Stock Class - this option allows you to change the stock class of all items within the batch to the class chosen. Note that the Stock Class change will only affect parts in the list which has a zero stock on hand value.

 

Created on - This option allows you to set the created date for stock items which currently have no created date on file. This is used for those customers who created stock in AM-Win prior to version 6. The created date can then be useful when running redundant stock reports.

 

Bin Location - This allows you to set the same bin location for all of the items within the batch.

 

Preferred Supplier - This allows you to set the same preferred supplier for all of the items within the batch. You may now purchase a particular group or groups of parts from a different supplier. This will allow you to change the supplier and from now on the Auto orders and other functions will work for those items from the new supplier.

 

Cost Ex and Cost Including GST. - This allows you to set the same preferred supplier for all of the items within the batch. You may find that all spark plugs for example of a certain type have the same price. This will allow you to set the price without manually going through each individual item.

 

Group - This allows you to allocate all of the parts within the batch to a specific group. These might be previously orphaned parts which had no group or they might be a combination of groups that you now wish to merge into a single group. You may also wish to split a group into several groups by generating a batch by group, deleting the unwanted items from the batch and then creating a new group with the remaining batch items.

 

Sub Group - This allows you to allocate all of the parts within the batch to a specific Sub group. These might be previously orphaned parts which had no Sub group or they might be a combination of groups, suppliers or other filters that you now wish to merge into a single Sub group. You may also wish to split a group into several sub groups by generating a batch by group, deleting the unwanted items from the batch and then creating a new sub group within the chosen group using the remaining batch items.

 

Cross Reference (X-ref) - This allows you to allocate all of the parts within the batch to a specific cross reference. This allows you to select similar parts with something in common and then cross reference them together for invoice, purchase or lookup options in AM-Win.

 

Auto Orders - This allows you to allocate all of the parts within the batch to a specific Auto Order status.

 

Status - This allows you to allocate all of the parts within the batch to a specific status of inactive or active. You may wish to make a whole group or stock based on group, subgroup or supplier inactive because you no longer use those items.

 

Category - This allows you to allocate all of the parts within the batch to a specific stock category.

 

Usual Qty - This allows you to allocate all of the parts within the batch to a specific quantity.

 

Code Has Prefix - This option allows you to change the status to Yes or No (Ticked or Unticked). If left blank then the status will remain unchanged.

 

Is Serialised - This option allows you to change the status to Yes or No (Ticked or Unticked). If left blank then the status will remain unchanged. Note that this change will only take place for items with zero stock on hand.

 

Sell By Volume - This option allows you to change the status to Yes or No (Ticked or Un ticked). If left blank then the status will remain unchanged.

 

Use Additional Description - This option allows you to change the status to Yes or No (Ticked or Un ticked). If left blank then the status will remain unchanged.

 

Ignore Stk Multiplier - This option allows you to change the status to Yes or No (Ticked or Un ticked). If left blank then the status will remain unchanged.

 

DonPrint Barcode - This option allows you to change the status to Yes or No (Ticked or Un ticked). If left blank then the status will remain unchanged.

 

No Discount - This option allows you to change the status to Yes or No (Ticked or Un ticked). If left blank then the status will remain unchanged.

 

Is Core Stock - This option allows you to change the status to Yes or No (Ticked or Un ticked). If left blank then the status will remain unchanged.

 

Minimum Stock Level - This option allows you to change the Minimum stock level for all items within the selected batch.

 

Maximum Stock Level - This option allows you to change the Maximum stock level for all items within the selected batch.

Maintain Level - This option allows you to change the maintain level for all items within the selected batch.

 

Bulk Rate - This option allows you to change the Bulk rate for all items within the selected batch.

 

Qty On Hand - This option allows you to change the Qty on hand for all items within the selected batch. Note that the Qty change will only affect parts in the list which has a zero stock on hand value.

 

Asset GL Code - This option allows you to change the Asset code for all items within the selected batch. Note that the Stock Class change will only affect parts in the list which has a zero stock on hand value.

 

Revenue Code - This option allows you to change the Revenue code for all items within the selected batch.

 

Expense Code - This option allows you to change the Expense code for all items within the selected batch.

 

Rounding - This option allows you to round the selling prices in the stock file to the nearest 1c, 5c or $1 for all items within the selected batch.

 

Unit Per - This option allows you to change the unit per type for all items within the selected batch. You can type the unit per or select a default option from the list.

 

Selling Prices - This allows you to set specific selling prices to all of the selling prices. If a selling price is left at zero then no changes will be made. Because of this you are currently not able to make a price equal to zero.

 

The Process Button

The Process button allows you to process the selected batch and make all of the option changes that have been set in the options area.

 

When you click on the Process Button, if you have records tagged in the batch list then you will be given the option to process tagged records only or all records within the batch. If you select tagged records only then only those items in the batch list that are tagged will be updated. Remember that any change options selected, which break the rules will not be updated.

 

The Code Change Option

The Code Change option allows you to change the code of all or selected items within the batch list. This can be done in several ways. Firstly you are able to add in a prefix (Add Prefix Suffix) at the beginning of the part number or in fact from any position within 10 characters of the beginning of the part number. You can also add in a suffix (Add Prefix Suffix) at the end of the part number or in fact from any position within 10 characters of the end of the part number. You are also able to add in a space after or before any character of the part number including at the beginning of the part.

 

Apart from adding characters you can also remove (Remove Prefix Suffix) up to 10 characters from anywhere in the part number.

 

Add a Prefix or a Suffix

When the Change Code tab is selected you have the choice to either add or remove. Select the Add option which is the default.

 

You then need to select either the Prefix or the Suffix option. Prefix means that the change will occur from left to right whereas suffix means the changes will occur from right to left.

 

You are then required to enter the position which the changes will start after. If the position is left at zero then the prefix will be added to the beginning of the code. If the position is left at zero and the suffix option is chosen then the suffix will be added at the end of the code. If the prefix is selected and the position is set to 2 then the prefix will be added after the second character in the part number where as it will be added before the second last character if the suffix option is chosen. The position has a maximum of 10.

 

The prefix / suffix character allows you to enter up to 5 characters to be added into the existing part number.

 

You can also tick the option which allows you to add a space at a given position in the part number. If you select Prefix and tick the space option and then select the space tick box the system will insert a space in the part number at the position selected in the space position.

 

You can then click on the TEST BUTTON to test that the changes you have requested or that you understand to have requested are what you are actually going to get. This will show you a sample of the first record in the batch list and how it will be affected by your requested changes. If you are happy with the requested changes you can then press the PROCESS button to update the changes. The changes made will be reflected in your sales, purchases, history and other areas of the software.

 

NOTE: This type of change done on a large batch of parts may take considerable time to process.

 

Removing a Prefix or a Suffix

When the Change Code tab is selected you have the choice to either add or remove. Select the Remove option.

 

You then need to select either the Prefix or the Suffix option. Prefix means that the change will occur from left to right whereas suffix means the changes will occur from right to left.

 

You are then required to enter the position which the changes will start after. If the position is left at zero then the removal will be started at the beginning of the code. If the position is left at zero and the suffix option is chosen then the removal will be taken from the end of the code. You are then required to enter the number of characters to remove. If you select position zero and characters 1 and the prefix option then the system will remove the first character from the part number. It will remove the last character (first from the right) if suffix is chosen. If you chose position 1 and then 2 characters and prefix the system will remove the 2nd and 3rd character from the part number. This being 2 characters after position 1. Simple huh?

 

You can then click on the TEST BUTTON to test that the changes you have requested or that you understand to have requested are what you are actually going to get.

 

This will show you a sample of the first record in the batch list and how it will be affected by your requested changes. If you are happy with the requested changes you can then press the PROCESS button to update the changes. The changes made will be reflected in your sales, purchases, history and other areas of the software.

 

NOTE: This type of change done on a large batch of parts may take considerable time to process.

 

The Price Change Tab

The Stock Data Tools module allows you to change parts pricing based on a percentage, dollar amount or a value range within a selected stock batch.

 

You must first create a New Batch or revise an existing batch to be used and then you can make price changes based on the selected options Price Change Options from the Price Change tab.

 

Price change - Percentage

Percentage - To change the price of the items in the batch by a percentage value click on the percentage option. You then are required to click on the Percentage Button to bring up the price increase window. You can then enter the percentage to be used for the price increase. Once this is done you can then select to increase either the cost or the selling prices by that percentage.

 

Increasing cost prices

The cost price increase option will increase the ex and Inc GST cost prices and the selling prices only. The average cost price will not be altered. Once you have selected the cost price, you have 3 further options to choose from.

 

Retain Cost/Sell Margin - This option will retain the existing margin between costs and sell price of the items. This means that the cost and selling prices will increase by the same percentage.

 

Absorb Cost Increase - This option means that the selling prices will remain the same even though the cost price increases.

 

Recover Cost Increase - This option means that the selling price will increase by the same dollar value as the cost price increases regardless of the percentage of the increase.

 

Click the OK Button to accept the options you have chosen. You can then click on the Process button to process the selected options for the stock batch. You can also select to process only the tagged items inside the Batch list.

 

Increasing Selling Prices

The selling price increase option will increase the selling prices only. Once you have selected the selling price, you can then click on the tick box next to the prices you wish to have increased. Only the ticked selling prices will be changed.

 

Click the OK Button to accept the options you have chosen. You can then click on the Process button to process the selected options for the stock batch. You can also select to process only the tagged items inside the Batch list.

 

Price Change Amount

Amount - To change the price of the items in the batch by a dollar value click on the amount option. You then are required to click on the Amount Button to bring up the price increase window. You can then enter the amount to be used for the price increase. Once this is done you can then select to increase either the cost or the selling prices by that percentage.

 

Increasing cost prices

The cost price increase option will increase the ex and Inc GST cost prices and the selling prices only. The average cost price will not be altered. Once you have selected the cost price, you have 3 further options to choose from.

 

Retain Cost/Sell Margin - This option will retain the existing margin between costs and sell price of the items. This means that the cost and selling prices will increase by the same percentage.

 

Absorb Cost Increase - This option means that the selling prices will remain the same even though the cost price increases.

 

Click the OK Button to accept the options you have chosen. You can then click on the Process button to process the selected options for the stock batch. You can also select to process only the tagged items inside the Batch list.

 

Increasing Selling Prices

The selling price increase option will increase the selling prices only. Once you have selected the selling price, you can then click on the tick box next to the prices you wish to have increased. Only the ticked selling prices will be changed.

 

Click the OK Button to accept the options you have chosen. You can then click on the Process button to process the selected options for the stock batch. You can also select to process only the tagged items inside the Batch list.

 

Price Change Range

Range - To change the price of the items in the batch by a range value click on the range option. You then are required to click on the Range Button to bring up the price increase window.

 

You can then select a choice of either a percentage or value to be used for the price increase. Once this is done you can then select to increase either the cost or the selling prices by that percentage.

 

Increasing cost prices

The cost price increase option will increase the ex and including GST cost prices and the selling prices only. The average cost price will not be altered. Once you have selected the cost price, you have 3 further options to choose from.

 

Retain Cost/Sell Margin - This option will retain the existing margin between costs and sell price of the items. This means that the cost and selling prices will increase by the same percentage.

 

Absorb Cost Increase - This option means that the selling prices will remain the same even though the cost price increases.

 

Recover Cost Increase - This option means that the selling price will increase by the same dollar value as the cost price increases regardless of the percentage of the increase.

 

The Range - Must be set for a specific dollar amount range for the changes to apply to. This means that you can select the first of the 3 ranges to be for example $0.00 to $2.00 with an increase of 50% and then the second range from $2.01 to $50 with an increase of 40% and then a range of $50.01 to $999,999 as 30%. This means that you can apply a higher mark up to items that are in a lower price bracket. For example you may charge globes and fuses with a much larger mark up due to their low value than you would more expensive items where you can afford to sell them at a lower margin.

 

Click the OK Button to accept the options you have chosen. You can then click on the Process button to process the selected options for the stock batch. You can also select to process only the tagged items inside the Batch list.

 

Increasing Selling Prices

The selling price increase option will increase the selling prices only. Once you have selected the selling price, you can then click on the tick box next to the prices you wish to have increased. Only the ticked selling prices will be changed.

 

The Range - Must be set for a specific dollar amount range for the changes to apply to. This means that you can select the first of the 3 ranges to be for example $0.00 to $2.00 with an increase of 50 cents and then the second range from $2.01 to $50 with an increase of $2 and then a range of $50.01 to $999,999 as $10.

 

This means that you can apply a different mark up to items that are in a different price bracket. For example you may charge globes and fuses at 50 cents more whereas more expensive items can be increased by a larger value.

 

Click the OK Button to accept the options you have chosen. You can then click on the Process button to process the selected options for the stock batch. You can also select to process only the tagged items inside the Batch list.

 

Merging Stock Files

There may be times when you have data in another AM-Win stock file which you wish to have merged into your main stock file. This may be when you are merging 2 companies into one set of data, merging your practice stock file into your live file or obtaining parts from an associated companydata. For whatever reason, the Stock Data Tool will allow you to merge another stock file into your main one. To perform this function you must first select the File Merge TAB from within the Stock Data Tool Module.

 

Merge Rules

 

  1. If a stock item exists in the other stock file but not your main file then the item will be added into your existing file. If you choose to add the quantity on hand then the new item will also show as in stock and a transaction will be processed to add the value of stock into your ledger as an opening balance record. This will be offset against the reserve account and the appropriate stock ledgers.

 

  1. If a stock item being merged exists in both the input file and your main stock file then the item will not be added a second time. However, if you choose to add the quantity on hand then the item stock on hand figures will be added together and a transaction will be processed to add the value of stock into your ledger. This will be offset against the reserve account and the appropriate stock ledgers.

 

  1. Only Stock items have a transaction added. Non Stock items do not add to the stock value.

 

Select Input File

The input file is the other stock file you wish to merge with your existing main stock file. To select the input file, click on the Select Input File button. This will then open the usually Windows file selection window. Select the file you wish to use as the input file (Not your main stock file) and double click on it. The input file location is displayed on the top right hand side of the screen for reference purposes.

 

Previewing the input file.

You can preview the first 100 records of the input file by clicking on the Preview tick box above the Input File button.

 

Previewing the Main (Output) file.

You can preview the entire output file by clicking on the Preview tick box next to the Output File label.

 

Input File Options

You have a choice of importing only stock items from the input file that belong to either a specific supplier or a specific stock group.

 

Filter by Supplier

To import only records from the input file belonging to a specific supplier, click on the supplier tick box area and then select the supplier to be used. You can then preview the first 100 matching records from the input file by ticking on the preview button.

Filter by Group

To import only records from the input file belonging to a specific group, click on the group tick box area and then select the group to be used. You can then preview the first 100 matching records from the input file by ticking on the preview button.

 

Stock on Hand

There is a tick box option which allows you to update the stock on hand figures between the input and the main stock file. IF you tick the Update Stock on hand option then the stock on hand from the input file will be merged with the main stock file. Stock will be added together if the item exists in both files. A transaction will also be written to allow for the increase in stock between the reserve account and the stock ledgers.

 

Process Merge Button

Once you have selected the input file, select input file, selected your merge options input file options and chosen to either update or not update the stock, you can then start the merge process by clicking on the Merge Button.

 

NOTE: that if you are importing a large stock file then this process may take considerable time.

 

The Undo Tab

The Undo tab allows you to undo the last change that was processed Process Button through the stock data tool. You may have just processed a change and then realised you made a mistake, had the wrong batch, updated the wrong group or prices etc.

You can then undo the last process that was done but only the last process and not any before it.

 

Click on the Undo Tab. You can then click on the Load Undo File button to load the parts that will be undone. You can then click on the process button to make the changes. You can only choose to undo a whole process. In other words, you cannot partly undo something. Itall or nothing.

 

Once the undo file has been loaded you can then click on the Process button to process the undo request.

 

Process Limitations.

You are not able to undo a process that was done more than 5 days ago even if that process was the last one done. This is because to many changes may have already been done to effect those stock items in the normal course of running AM-Win for an undo to be considered safe.

 

You can only undo the last process that was completed. You cannot undo the one before etc. This is because the last process may have already changed the process that went before it.
