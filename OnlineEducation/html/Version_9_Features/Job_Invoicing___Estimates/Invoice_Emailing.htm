Invoice Emailing

In the customer (and supplier) masterfiles there is now an option to flag a customer (or supplier) for their invoices and other documents to be emailed by default rather than to be printed and sent by mail or fax. If the tick box is ticked then when an invoice or other document is printed, it will create a PDF file and attach that PDF file to an email using your default email program. You can then add comments to the email if required and send the email in the usual way. The system uses the first email address from the masterfile but you can change the address if required prior to sending the email.

You can also select the Email option from the print options if you decide at a later date to email a document rather than print it.
