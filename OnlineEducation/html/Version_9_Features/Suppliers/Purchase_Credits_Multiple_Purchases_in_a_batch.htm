Purchase / Credits

Multiple Purchases in a batch

 

Purchase credits is an alternative to Purchase Orders. The difference lies in the fact that purchase credits allow purchases to be batched in for a particular supplier for multiple invoices to the suppliers. However, purchase orders only allow one purchase to be done at a time. Purchase credits is also a quicker option if orders are not required to be printed.

You are also able to use purchase credits to batch purchase invoices from the same supplier. You do this by simply changing the date and reference number of the invoice allowing multiple invoices to be processed in a single entry. The system will then on processing, automatically consolidate the individual lines into a separate invoice for each individual reference number. You can also see the total value of a certain invoice prior to posting the invoices by right clicking on the invoice line and selecting the Ref Tot option.
