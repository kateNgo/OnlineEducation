GST Payments

There is now an option to enter your GST payments in AMWIN. There is no need for a manual journal entry to be done. Simply enter the date, cheque, method, Description and GST amount, installment income if any, PAYG amount, and any fringe benefits tax into the relevant fields and the system will ensure your payment balances to the correct ledgers by automatically doing the journals required to update the ledger and Bank Rec.

Go to the Ledger menu and choose GST Payments. The GST Payments window will open.

Date: click on the button next to this field and select the date by clicking onto the calender which will appear

Refer/Cheque: type in the reference or cheque number

Method: click the arrow next to this method field and select the method ie bank, cash.

Description: type in the appropriate description

Amount: type in the total amount which is to be paid.

Please note: if the amount you are entering is payable to the ATO this value must be a Positive Amount. If the value is a refund then this value must be a Negative amount.

Within the GST, PAYG, Instalment fields, simply type in the amount allocated to these accounts. Please make sure you have set up accounts for GST, PAYG and Instalment in your general ledger masterfiles.

For the Fringe Benefits field, click on the ledger button next to the blank field and select the ledger account you want the fringe benefits tax to go into.

As long as the Left to Post = Zero, you may click the Process button to process this GST payment.
