New Quill Function

 

When using the Quill function in Job invoicing there is now a new feature that allows you to edit text using the quill. If you select the Quill Button while highlighting a line of text, you are given a button to either Change the selected text, Insert new text or to Cancel the operation. This choice is only given if a line of detail is highlighted. Otherwise, it will automatically assume the insert mode.

 

If the change option is chosen then the quill will be loaded with the line of text highlighted PLUS all lines of text below it until it reaches a line which is not a text line. This means that if you have 5 lines of text following by a part or labour line and you highlight the second line of text and use the quill change option, the 2nd  the 5th line of text is loaded into the quill notes. You can then alter, add to or delete from or move around the notes inside the quill and then place them back onto the invoice.
