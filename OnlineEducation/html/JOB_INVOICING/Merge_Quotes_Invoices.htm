Merge Quotes & Invoices

 

Quotes / Estimates are incorporated into the standard job and workshop invoicing modules. There is no longer a separate quote / estimate option. This allows all of the features found in job invoicing to be utilised when creating estimates.

 

When you enter the job invoice or workshop invoice screen there is a selection box that allows you to choose either Job or Quote. The default is Job. If left as job then the job will act as it always has in invoicing. If the option is changed to Quote then a quote will be produced instead of a job. The functions to add parts, labour, revenue, detail etc to a job and quote are the same. All of the functions such as reprice, subtotals, cut and paste etc remain the same.

 

The Differences.

 

Jobs and Quotes are now stored in the same file together but are separated by a field in the system which determines what they are. The system will know when to display jobs and when to display invoices.

 

If you have selected a quote then you have the opportunity to select the number of days the quote is valid for which is printed on the quote. You are also able to label the quote in whatever manner you wish. Such as a label of QUOTE or ESTIMATE or CHECK or WARRANTY etc and that will be the document heading. For an invoice the document heading is always TAX INVOICE.

 

Quotes and Jobs use different numbering systems so that Jobs and Quotes can use sequential numbers within the same document type. While a Job number can be manually assigned by the user this is NOT the case of Quote numbers which are always automatically assigned by the system.

 

A quote cannot be finalised as it is not a financial document which updates the ledgers, customers etc. It can however be revised, deleted or turned into a job.

 

A job card cannot be printed if the document is a Quote and the Proforma Invoice button becomes the quote print button.

 

Parts cannot be allocated to a quote unless that quote is turned into a job first.

 

Quotes can be manually deleted from the system one by one or you can use the new Quote deletion tool which is located under the others menu in AM-Win to delete a range of quotes between dates if they have expired.

 

When revising a Job or Quote there is a separate TAB that you must use to select either a Job or Quote to be revised.

 

Quotes > Jobs or visa versa

 

A Quote can be converted to a Job at any time and a Job can also be converted to a Quote at any time.

 

New Job / Quote

 

If you start a new Job or Quote and have not yet saved the document then you may change it to either document type and the system will treat it as the selected document automatically and then assign the appropriate document number.

 

Previously saved Job / Quote 

 

If a Job or Quote has been previously saved then you can convert that document to the alternate document type by changing the document type on the screen. NOTE: You can only change a document type immediately after it has been saved unless it is a document that has not previously ever been saved. You must first save the document using the save button, then change the document type and then click the save button again. If you save the document as a different document type then a message will appear giving you three choices. These choices are Create, Replace or Abort.

 

Update

 

This option allows you to create the document type you have selected and will leave the existing document on file for future reference. EG: If you create a new job from an existing quote then the quote will remain on file and a clone of that quote will be created as a new job.

 

Delete

 

This option will create a new document type and remove the existing document. EG: If you replace a quote with a job then the quote will be deleted and a cloned new job will replace it.

 

Retain

This option will create a new document type and retain the existing document as it was previously saved.

 

Abort

 

This option will return you back to the screen and nothing will be updated. The document type will be returned back to the original document type of the saved record.

 

Cloning a History job to a New Quote

 

When cloning an invoice from the History module you can select to clone the Job into either a Quote or into an Invoice by selecting the document type.
