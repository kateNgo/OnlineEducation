Insert Stock
Choose Parts  F3 button or press F3 key to insert parts in the record. The Select Stock or Alternate Stock window appears. Enter Stock Item Code and or left click to highlight the stock item, then choose Select to confirm the selected part.

 

 Note: Choose Search to search for a Stock Code.

 Choose Insert to insert a new stock.

 

When a Stock Code has been entered/selected, the Insert Part Record window appears.

 

Discount Rate - Enter percentage allowed for Stock discount for this customer, if applicable. This information comes from the discount field in the invoice header section of the screen but can be changed for individual Stock. You can also use the discount matrix if you want different groups of Stock to have different discount levels. Such as Brakes 30% off, Exhausts 40% off and engine parts 10% off etc.

 

Qty  Enter the number of parts to be used or supplied.

 

Price  Enter the Price per part. This price is automatically selected from the stock masterfiles pricing depending on the price rate set up for this customer which is set in the customer masterfiles. This price can be manually overridden for each item.

 

Under the passwords option from the Control System, there is a new password level that determines what level a user is required to have to alter the price of a part and the discount level of a customer from the invoicing screen. If the user has a password level that allows them to enter parts on an invoice but not a sufficient level to change prices then this ensures that the user charges the prices from the stock file and cant over ride them to a cheaper price.

 

Units  Unit of Stock item (automatically inserted). These come from the stock masterfiles and are texts such as EACH, SET, PAIR etc.

 

Bin No - the location of the stock number eg. Bin would mean that this part is stored in the row and is in the 12th bin. (automatically inserted) from the stock item record.

 

Extension  The total price of the number of parts (automatically calculated). You can change the extension and it will automatically recalculate the unit price.

NOTE: If the stock item is marked as being sold by Volume (Usually petrol or items sold by litre or weight) the changing the extension will result in the qty being recalculated rather than the unit price.

 

Less Discount  The amount to be discounted (automatically inserted). Calculated from the discount rate.

 

GST  The amount of GST that the part attracts (automatically calculated). This rate is set from the control system. If this is an export invoice then the rate will be zero. It will also be zero if the stock item is set to be GST exempt in the stock masterfiles.

 

Total  The total amount to be invoiced for the part, less any discount.

 

Note:  The total amount displayed includes the GST. Do not try to take off the discount and then add the GST as you will not get the correct result. The total is the qty * unit price less discount and the GST is calculated from the total.

 

C  Left click on the C button to view the cost price of the part per unit.

 

S  Left click on the S button to view the Stock Masterfiles for the selected part number.

 

Reprice  F5 - This button allows the user to permanently change the price in the stock masterfiles for the selected item to a new price charged for this invoice. If you manually override the stock price, this will save the new price. NOTE: If the customer is on price rate one it only changes price 1 in stock. If the customer is on price rate 2 then it only changes the price in rate 2 etc.

 

Choose More, if another part record is to be added and repeat the insert stock procedure. You can also press the ENTER key which acts as a more button to add further items.

If the select stock item is a serialised stock item then read the help for serialised stock.

 

Choose OK or the F12 key, if there are no more parts to be added to the record.
