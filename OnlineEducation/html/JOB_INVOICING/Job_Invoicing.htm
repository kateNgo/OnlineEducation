JOB INVOICING

 

This option is used to create service / workshop / repair invoices and to retain a history on file for invoices, for revising invoices and to book in jobs. It is suggested that this option can be used for the service and warranty type invoices. A Future Service Log can also be updated using this invoice. Any job relating to servicing or repairs should be done through this section. It also allows export invoices to be created for overseas sales.
