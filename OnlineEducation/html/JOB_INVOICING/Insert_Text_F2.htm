Insert Text  Alt T

 

Comments / Text allows up to 10 lines of detail to be retrieved from each record. This is great for standard job stories such as Tune ups, Oil Change, Brake Service and other job details that would otherwise have to be manually typed. This is a great time saving option.

 

To insert text from the comment library, select Text  Alt T or press the Alt T keys. The Select Text window appears:

 

Left click on the Text Code so that it is highlighted, and choose Select. The

comment will be automatically inserted into the job invoice. By highlighting this text within the invoice and clicking on the Delete button, you may remove this line of text.

 

Note : Comments can viewed by Browsing the Comments Library.

 

When importing standard text comments into a document you can see the content of the text you are importing so that you donmake the mistake of importing the wrong text which has a similar name.
