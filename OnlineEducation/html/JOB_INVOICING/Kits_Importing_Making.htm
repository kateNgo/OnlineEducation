Kits Importing / Making

 

Importing / Making Kits  Alt K

 

To import pre-formatted invoices which contain comments for a particular

customer or for a particular job, choose Kit  Alt K or press the Alt K keys. The Import and Export Kits window appears. The Kit is like a completed job or service that can be imported into the current invoice.

 

Kit Code  Enter kit code to search for the required kit.

 

Import Kit

 

Left click on the kit and this highlights the kit to be imported. Click on the Import Button. This automatically inserts the Kit code in the Import and Export Kits window. Select OK to confirm. The details from the kit will be automatically inserted into the job invoice. The parts and labour items in the Kit are automatically repriced according to the customers price rate and discount rates if the reprice option is ticked.

 

New Kit

 

To make a new kit from the existing details in the invoice that is currently in use Type in the new kit code and kit description and then click on the New Kit button. This kit is now available to be imported into other jobs.

 

You can also make kits from the Job History module from jobs that have been previously finalised.
