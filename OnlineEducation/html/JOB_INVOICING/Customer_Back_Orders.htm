Customer Back Orders

 

During the process of invoicing parts to a customer you quite often are not able to supply all of the parts required. In this case you need to order the parts. When you start an invoice and then select the option to place parts onto that invoice you get the option to automatically order any parts required while remaining in the invoice module. When you go to select a part, once the part is select and you are at the point where you can change the Qty or Price to charge there is a small button marked with the Letter A which stands for Allocate.

 

If you click on the A button you are taken to another screen at which time you can order the selected part. If the selected part has a valid preferred supplier then the supplier will Automatically be filled in at the top of the screen. You are then able to change the supplier if you wish to order the part from somewhere else. If no preferred supplier was selected then you are required to select the supplier.

 

The system now displays all currently outstanding orders for the selected supplier in the top left-hand list box. Another list box beneath that shows all outstanding orders that exist for the selected customer. A further list box to the right of that displays all outstanding orders for the selected part. This means you can see if the selected part is already on order to avoid ordering additional items that are not required.

 

You are now able to place the order for the selected item or otherwise to exit the screen if you choose not to place an order for this item. If you decide to place the order then you have 2 options. You can either add the part to an existing order for the selected supplier or to create a new order to that supplier. If you wish to add the part to an existing order (Say a daily purchase order which gets appended during the day and then printed after lunch) then you must first click once on that order in the suppliers outstanding orders list at the top left of the screen. This will highlight the order.

 

You then need to enter the quantity to order and if you wish you can alter the order date and enter in the description of the order (such as Daily Order). If this was done through a Job Invoice and you had already saved the job to generate a job number then the job number will also appear on the screen.

 

NOTE! If you have not saved the Job then the order system will not be able to tell you what job number these parts were allocated to.

 

You are now able to select either of two process buttons. You can press the F9 key or click on the New Order button to generate a new purchase order to the selected supplier for the selected part. Alternatively you can press the F10 key or click on the Selected button to add the selected part to the currently selected purchase order (say the daily order).

 

NOTE!  You are also able to operate the customer orders module from the Customer menu option.

 

The part will now automatically be on either the appended or new purchase order. When the item is delivered on the Purchase order screen you can print a delivery slip which tells you what customer and also what job (if applicable) the order was processed for.
