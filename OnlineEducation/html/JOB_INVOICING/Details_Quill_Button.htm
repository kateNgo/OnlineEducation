Details - Quill Button

 

Import Notes into Job Invoice

 

In Job Invoicing, Workshop Invoicing, Credit returns and Purchase orders you are able to import or type details into an invoice in a similar way that you would type in a basic wordprocessor. There is a button with a icon of a quill in it that operates this function from within job invoicing, Credit returns and Purchase orders, and it is a standard detail function in Workshop invoicing.

 

Manually typing text using the notes option.

 

Select the quill button and a notes window will appear which allows you to type information just like a basic wordprocessor would. You can then type in up to 6000 characters of information which equates to up to 100 lines on an invoice, credit, purchase order etc. If you hit the enter key at any point in a line you are typing then that line is treated as a separate line on the document. Otherwise if you simply continue to type then the software will format the lines as best it can for you. You can view how the lines will look on the document at the bottom of the window by pressing the F6 key or clicking on the Preview - F6 button. If you are happy with how it will look then click the ok button or press the F12 key and the text will be transferred to your document (invoice etc). If you are not happy with it then change the details in the notes window until you are.

 

Changing the Fonts

 

When in the notes window you can select the Fonts button by oressing the F4 key or by clicking on the button. You can then select the font which will be applied to all of the text including the colour, style, type and size. Note that you can also use the * button to obtain the usual heading font. (See the invoice section of the full manual for the correct use of the * option.

 

Importing Job Item Notes to an invoice.

 

If you are using the notes option from Job or Workshop Invoicing then you can transfer the details stored in the job item notepad directly into the notes window and then into the invoice. You can also change the details or fonts before importing the text to the invoice.

 

Importing notes from an external source.

 

The copy and paste commands built into windows enables you to copy information from external sources such as an email, a website, a word or other text document, notes from another program such as a technical specifications database etc directly into an invoice through the notes window. To do this, you highlight the text from the external source you wish to use and then press the CTRL & C buttons together. This will copy the external text into an area of memory used by windows to store such information. You then go into the notes window and use the CTRL & V buttons together which pastes the external text into the notes window. You then can change, preview and put the text onto your document. This is great for transferring from times guides or if a customer sends you email instructions that you want to paste onto a job rather than retyping all the details.
