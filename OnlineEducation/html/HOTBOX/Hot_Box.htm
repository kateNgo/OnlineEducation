Hot Box

 

!! HOT BOX !!

 

Description.

 

The Hot Box is a menu selection box that is specific to each computer. This means that the Hot Box menu could be (and most likely is) different for every computer on your network (if you have more than one computer connected).

 

The idea of the Hot Box is that each computer terminal may be used for a different purpose. The accounts computer, spare parts computer and workshop terminal all get used for a different purpose. For example the accounts computer does accounting transactions, prints accounting reports and updates the ledger entries whereas the spare parts computers sell parts, updates stock, does price adjustments etc and the workshop computer books in jobs, checks service history and looks up job item information.

 

The HOT BOX is activated from anywhere within AM-Win by pressing the CTRL-H buttons together or by selecting the HOT BOX from the main menu option. (CTRL  H is the preferred and quickest method).

 

Setting it up

 

To set up your HOT BOX menu you must first go into the Update Hot Box menu located under the main Hot Box menu option. If the options on the left of the Hot Box update screen are blank then you will need to contact your AM-Win support agent to obtain the file required to set up the Hot Box menu.

 

To add menu options to your Hot Box menu you must highlight the required option on the left of the screen and then click the Add One button which places the selected menu item onto the right of the Hot Box menu. Repeat this step for each new item. The Hot Box is meant to be a quick selection menu of your most used menu choices. Please do not over populate the Hot Box as this will defeat the purpose of a Quick and Easy menu. A maximum of 10  15 menu items is recommended.

 

You can also move items into the Hot box menu by dragging them from the left of the screen into the right of the screen using the mouse. This is done by highlighting the record on the left and whilst holding down the left mouse button, drag the mouse (and the menu item) over to the right side of the screen and then release the mouse button.

 

You can also remove an option be highlighting it on the right of the screen and then clicking on the remove one button. You can also drag the option from the right of the screen to the left of the screen as explained in the previous paragraph.

 

By selecting the remove all button it will clear the Hot Box menu.

 

You can now exit the screen using the exit button or the escape key and your Hot Box menu is ready to use.

 

 

Using the HOT BOX

 

Now you have set up the Hot Box as per the instructions above (do it now if its not already set up) you can use it. To access the Hot Box from anywhere within AM-Win (not from additional modules) you simply press the CTRL-H keys. Do this by holding down the CTRL Key and then pressing the H button and then releasing the CTRL key.

 

The Hot Box menu will now appear. Use the arrow keys or the mouse to select the required menu option and then either press the Enter key or double click the mouse to run the selected item. The menu is designed to be used without a mouse for maximum speed.

 

Colour Coding

 

The Hot Box is colour coded for ease of use and making it easier to recognise your menu selections. All customer selections are coded in Blue, Suppliers in red, stock in green, Floor Plan in dark blue, Ledger in maroon, Reports in Purple and enquiries in Black. You can click on the colour code boxes to select the first menu option for that type of selection. For example if you click on stock the first stock menu option will be highlighted.
